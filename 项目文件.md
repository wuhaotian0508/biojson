#### 1.1 项目概览 （suppose 每周重复）

- 项目背景：植物营养代谢领域的核心基因信息分散在大量科学文献中，人工整理耗时且易遗漏。BioJSON 项目旨在利用 LLM（大语言模型）从论文中自动提取基因-代谢产物-调控关系等结构化信息，并通过二次验证降低 LLM 幻觉，最终沉淀为可复用的知识资产。
- 关键科学问题 / 技术问题：（1）如何让 LLM 准确提取 40+ 个生物学字段，并根据基因功能类别（通路酶 Pathway / 调控因子 Regulation / 通用 Common）自动选择对应的提取模板？（2）如何系统性地检测和修正 LLM 幻觉，实现字段级的忠实度验证？（3）如何将提取结果高效组织为可查询、可检索的知识库，支持 RAG 问答和专家标注审核？
- 项目目标：（1）建立从论文 Markdown → 结构化 JSON 的自动化提取-验证管道（Pipeline），目前已处理数十篇论文、提取 200+ 个基因；（2）搭建 RAG 基因信息问答系统，支持基于向量检索 + LLM 生成的结构化问答；（3）搭建 Web 标注平台，供研究人员对 AI 提取结果进行字段级审核与修正
