### 核心功能展示

1. **流式输出**
   - 实时显示LLM生成的答案
   - 支持SSE（Server-Sent Events）

2. **来源标注**
   - 每条信息自动标注 `[文章名 | 基因名]`
   - 点击可高亮显示

3. **参考文献列表**
   - 显示检索到的相关文献
   - 包含相关度评分
   - 最多显示10条

4. **会话历史**
   - 自动保存最近20条查询
   - 点击可快速重新提问
   - localStorage本地存储

## ✅ 已验证的测试问题

### 1. 基因功能查询

**问题**: NAC37基因在植物中有什么作用？

**预期回答要点**:
- NAC37是NAC家族转录因子
- 参与花青素生物合成调控
- 与LBD4协同激活Bz1启动子
- 对不同启动子具有特异性调控
- 与LBD4存在物理互作

**来源标注示例**:
```
[A Maize Gene Regulatory Network for Phenolic Metabolism | NAC37]
```

### 2. 代谢途径协同

**问题**: 哪些基因参与了花青素的生物合成？它们是如何协同工作的？

**预期回答要点**:
- EREB12 + ZHD14 组合激活Bz1
- NAC37 + LBD4 组合调控Bz1和A1
- 组合式转录激活机制
- 蛋白物理互作
- 启动子特异性调控

**来源标注示例**:
```
[A Maize Gene Regulatory Network for Phenolic Metabolism | EREB12]
[A Maize Gene Regulatory Network for Phenolic Metabolism | ZHD14]
```
