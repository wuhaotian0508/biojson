# Interaction of ${ \mathsf { Y } } { \mathsf { - F } } { \mathsf { e } } _ { 2 } { \mathsf { O } } _ { 3 }$ nanoparticles with Citrus maxima leaves and the corresponding physiological efects via foliar application

Jing Hu1†, Huiyuan Guo3†, Junli Li1,3*, Yunqiang Wang2 , Lian Xiao1 and Baoshan Xing3 

# Abstract

Background: Nutrient-containing nanomaterials have been developed as fertilizers to foster plant growth and agricultural yield through root applications. However, if applied through leaves, how these nanomaterials, e.g. γ $\mathrm { \Gamma } \mathrm { \bar { F } } \mathrm { e } _ { 2 } \mathrm { O } _ { 3 }$ nanoparticles (NPs), infuence the plant growth and health are largely unknown. This study is aimed to assess the efects of foliar-applied γ- ${ \cdot } \mathsf { F e } _ { 2 } \mathsf { O } _ { 3 }$ NPs and their ionic counterparts on plant physiology of Citrus maxima and the associated mechanisms. 

Results: No signifcant changes of chlorophyll content and root activity were observed upon the exposure of 20–100 mg/L γ-Fe2O3 NPs and $\mathsf { F e } ^ { 3 + }$ . In C. maxima roots, no oxidative stress occurred under all Fe treatments. In the shoots, 20 and 50 mg/L γ-Fe2O3 NPs did not induce oxidative stress while 100 mg/L γ- ${ \mathsf { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs did. Furthermore, there was a positive correlation between the dosages of γ-Fe2O3 NPs and $\mathsf { F e } ^ { 3 + }$ and iron accumulation in shoots. However, the accumulated iron in shoots was not translocated down to roots. We observed down-regulation of ferric-chelate reductase (FRO2) gene expression exposed to γ-Fe2O3 NPs and $\mathsf { F e } ^ { 3 + }$ treatments. The gene expression of a Fe2+ transporter, Nramp3, was down regulated as well under ${ \mathsf { Y } } { \mathsf { - F } } { \mathsf { e } } _ { 2 } { \mathsf { O } } _ { 3 }$ NPs exposure. Although 100 mg/L γ-Fe O NPs and $2 0 { - } 1 0 0 \mathrm { m g / L } \mathsf { F e } ^ { 3 + } | \epsilon$ $\mathsf { F e } ^ { 3 + }$ ed to higher wax content, genes associated with wax formation (WIN1) and transport (ABCG12) were downregulated or unchanged compared to the control. 

Conclusions: Our results showed that both γ-Fe O NPs and $\mathsf { F e } ^ { 3 + }$ exposure via foliar spray had an inconsequential efect on plant growth, but ${ \mathsf { Y } } { \mathsf { - F } } { \mathsf { e } } _ { 2 } { \mathsf { O } } _ { 3 }$ NPs can reduce nutrient loss due to their the strong adsorption ability. C. maxima plants exposed to γ- ${ \cdot } \mathsf { F e } _ { 2 } \mathsf { O } _ { 3 }$ NPs and $\mathsf { F e } ^ { 3 + }$ were in iron-replete status. Moreover, the biosynthesis and transport of wax is a collaborative and multigene controlled process. This study compared the various efects of γ-Fe O NPs, $\mathsf { F e } ^ { 3 + }$ and Fe chelate and exhibited the advantages of NPs as a foliar fertilizer, laying the foundation for the future applications of nutrient-containing nanomaterials in agriculture and horticulture. 

Keywords: ${ \mathsf { Y } } { \mathsf { - F } } { \mathsf { e } } _ { 2 } { \mathsf { O } } _ { 3 }$ nanoparticles, Nano-enabled fertilizer, Foliar spray, Wax, Gene expression 

# Background

Iron defciency in plants is widespread and can lead to reduction in crop yields and even complete crop failure [1]. Due to rapid conversion of iron into plant-unavailable 

forms when applied to calcareous soils, soil application of inorganic iron fertilizers to Fe-defcient soils is usually inefective [2]. In comparison, synthetic Fe-chelates for amelioration of iron defciency in plants is more efective, but more uneconomical [3]. It was reported that most foliar-applied micronutrients are not efciently transported toward roots, which may remain defcient [2]. Nowadays, nanomaterials become a hotspot of research interests and attract the attention of many researchers. 

A variety of nanoparticles (NPs) have been studied on human cells [4, 5], animal cells [6] and plants [7] about their toxicity or applications. As one of the most widely explored and applied nanomaterials, iron oxide nanoparticles $\left( \gamma { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 } \ \mathrm { N P s } \right)$ are widely used in medical diagnostics, controlled drug release, separation technologies and environmental engineering [8]. Iron dynamically released from $\gamma { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 } \mathrm { N P s }$ may be a potential nutritional source for plants. It is likely that $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs could be an efective fertilizer for alleviation of Fe-defciency in plants. Several studies have reported that root applied $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs have positive efects on plant growth. For instance, $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs can physiologically enhance seed germination, root growth, chlorophyll content in watermelon (Citrullus lanatus) planted in quartz sand [9] and Chinese mung bean (Vigna radiata L.) grown in silica sediment [10]. Rui et  al. [11] reported that $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs increased root length, plant height, biomass, and chlorophyll levels of peanut (Arachis hypogaea) plants, indicating that $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs can possibly replace traditional iron fertilizers in the cultivation of peanut plants. To our knowledge, few researchers reported the efects of $\gamma { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 } \mathrm { N P s }$ on plants via foliar application yet. 

Root is the major pathway for plants to absorb water and inorganic ions [12], through which NPs can be taken up and translocated to upper tissues [13–15]. When NPs were exposed to plants’ leave surface, several studies have observed that plants can absorb NPs through the leaves as well. Corredor et al. [16] reported that carbon coated iron NPs were capable of penetrating pumpkin (Cucurbita pepo L.) leaves and migrating to other plant tissues. Larue et al. [17] found that Ag NPs were efectively trapped on lettuce (Lactuca sativa) leaves and taken up by cells after foliar exposure. It is hypothesized that there are two pathways for leaves to take up NPs and their solutes: for hydrophilic compounds via aqueous pores of the cuticle and stomata, and for lipophilic ones by difusion through the cuticle [17]. Since the wax lipids may quickly adsorb on the large surface of NPs [18], particles might be trapped by the cuticular wax and then difuse in the leaf tissue (after dissolution or translocation through the cuticle) [19]. For example, Birbaum et  al. [18] reported that large agglomerates were trapped on the surface wax, whereas smaller particles might be taken up by the leaf. At the molecular level, wax inducer1 (WIN1), an ethylene response factor-type transcription factor, can activate wax deposition in overexpressing plants and infuence wax accumulation through the direct or indirect regulation of metabolic pathway genes [20]. Alabdallat et  al. [21] reported that WIN1 gene could modulate wax accumulation and enhance drought tolerance in tomato (Solanum lycopersicum) plants. Several plant ATP-binding 

cassette sub-family G member (ABCG) proteins are known or suspected to be involved in synthesis of extracellular barriers, among which ABCG12 is required for lipid export from the epidermis to the protective cuticle [22]. Te interactions between $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs and plant leaves were inevitably afected by cuticular wax due to the fact that the plant cuticles form the outermost barrier between plant leaves and their local environment. Terefore, it is of great signifcance to study the changes of cuticular wax in plant leaves induced by foliar sprayed $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs. However, to our knowledge, the efects of foliar application of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs on cuticular wax loads and related gene expression have not been reported. In the present study, in order to show the in-depth interactions between $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs and cuticular waxes in Citrus maxima leaves, wax content and wax synthesis or transport related genes, including WIN1 and ABCG12 were analyzed at the molecular level. 

Additionally, in order to fgure out the efects of $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs on plant growth and physiology, the corresponding parameters, including biomass, chlorophyll, soluble protein content, root activity, lipid peroxidation and activity of antioxidant enzymes were measured. C. maxima plants were exposed to 20, 50 and $1 0 0 ~ \mathrm { { m g / L } }$ $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs or $\mathrm { F e } ^ { 3 + }$ by foliar application at an early growth stage. Te latter treatment was set to study the phytotoxicity of $\mathrm { F e } ^ { 3 + }$ ions by dissolving $\mathrm { F e C l } _ { 3 } { \cdot } 6 \mathrm { H } _ { 2 } \mathrm { O }$ . Tis is the frst report on the $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs uptake and translocation in plants via foliar application, and the transcriptional modulation of genes involved in iron uptake or transport viz. ferric-chelate reductase (FRO2) and natural resistance-associated macrophage protein (Nramp3). 

# Methods

# Materials and experimental setups

Te $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs of $9 9 . 5 \%$ purity were purchased from Macklin Inc. (Shanghai, China). Te shape and size were determined by a Tecnai G2 20 TWIN transmission electron microscope (FEI, USA). Te hydrodynamic diameter and zeta potential were determined by a Zetasizer Nano ZS90 dynamic light scattering spectrometer (Malvern Instruments Ltd., United Kingdom). Te characteristics of $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs are shown in Additional fle  1: Figure S1 of the supplementary materials. $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs are spherical with an average diameter size of $2 0 . 2 \pm 2 . 7 \ \mathrm { n m }$ (Additional fle  1: Figure S1A). Te average hydrodynamic diameter and the zeta potential of $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs were $1 6 4 . 5 \pm 1 1 . 3 \mathrm { { n m } }$ and $- 1 1 . 7 \pm 0 . 1 \ \mathrm { m V } ,$ , respectively (Additional fle  1: Figure S1B, C). Citrus maxima seeds were immersed in distilled water and germinated in moist perlite at $2 8 ~ ^ { \circ } \mathrm { C }$ . Ten the uniform seedlings were transferred to a hydroponic system amended with 1/2 

Hoagland’s nutrient solution without iron. 18 of seedlings were planted in each hydroponic container. Plants were sprayed with $5 0 ~ \mathrm { m L }$ of deionized water (control), 20, 50 and 100  mg/L γ-Fe O NPs suspended in deionized water, 20, 50 and $1 0 0 \ \mathrm { m g / L } \ \mathrm { F e } ^ { 3 + }$ $\mathrm { F e } ^ { 3 + }$ (dissolved from $\mathrm { F e C l } _ { 3 } { \cdot } 6 \mathrm { H } _ { 2 } \mathrm { O } )$ solutions, and $5 0 ~ \mu \mathrm { M }$ Fe(II)-EDTA in the morning. During all the treatments, An iron-defcient control and a Fe(II)-EDTA treatment were set up for comparison. Te concentrations of $\mathrm { F e } ^ { 3 + }$ are calculated according to the containing iron content of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs at same concentration. Terefore, $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs and $\mathrm { F e } ^ { 3 + }$ treatments marked with the same concentration denote they have same iron content. Suspensions were sprayed with a hand-held sprayer bottle every 5  days when the plants had two true leaves. To facilitate foliar infltration, all plants were sprayed with deionized water once per hour for $^ { 1 0 \mathrm { ~ h ~ } }$ to avoid early evaporation of the solutions and consequent precipitation of solutes on the leaf surface [23]. Te plants were grown in an environmentally controlled growth chamber at $2 8 / 1 8 ~ ^ { \circ } \mathrm { C }$ with a $1 6 \mathrm { ~ h ~ } / 8 \mathrm { ~ h ~ }$ light/dark cycle; the light intensity was 2000  lx. Te air was pumped into the hydroponic system every $^ { 3 \mathrm { ~ h ~ } }$ with $3 0 \ \mathrm { m i n }$ each time. Te nutrient solution was replaced every 5  days. After 30  days of exposure, representative parameters including chlorophyll, fresh biomass, soluble protein content, root activity, lipid peroxidation, antioxidant enzyme activities, iron content, iron-related gene expression, wax content, and wax-related gene expression were measured. 

# Fresh biomass measurement

Citrus maxima plants were carefully removed from the hydroponic system after 30  days. Te fresh biomass of C. maxima including roots and shoots was weighed by using a FA1004C electronic analytical balance (Shanghai Yueping Scientifc Instrument Co., Ltd, China). 

# Measurement of physiological and biochemical parameters

Chlorophyll content was determined by a modifed procedure according to Lichtenthaler [24]. Soluble protein content was estimated according to a dying method using Coomasie Brilliant Fluka G-250 [10]. Measurement of root activity was according to the triphenyltetrazolium chloride method [25]. Malonaldehyde (MDA) was determined by the thiobarbituric acid method according to Heath and Packer [26]. Te activity of superoxide dismutase (SOD) was evaluated by the ability to inhibit photochemical reduction of nitroblue tetrazolium according to Wang et  al. [27]. Te activity of catalase (CAT) was analyzed as described by Gallego et al. [28]. Te activity of peroxidase (POD) was estimated by guaiacol colorimetric method as described by Zhang et al. [29]. 

# Metal uptake analysis

Harvested leaf tissue was rinsed with deionized $\mathrm { H } _ { 2 } \mathrm { O }$ thrice to remove the surface retained $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs. All shoot and root samples were dried at $6 0 ~ ^ { \circ } \mathrm { C }$ for $4 8 \ \mathrm { h }$ in a drying oven. $1 0 0 ~ \mathrm { { m g } }$ of oven-dried shoot and root tissues were separately digested in $3 ~ \mathrm { m L }$ of concentrated $\mathrm { H N O } _ { 3 }$ at $1 1 5 ~ ^ { \circ } \mathrm { C }$ on a hot block for 1  h. After cooling to room temperature, $0 . 5 \mathrm { m L }$ of $3 0 \% \mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ was added to the digestions at $1 0 0 ~ ^ { \circ } \mathrm { C }$ for $0 . 5 \mathrm { ~ h ~ }$ . Te iron content was analyzed by an Avanta M atomic absorption spectrophotometer (GBC, Australia). 

# Measurement of wax loads

Te content of cuticular waxes was determined using chloroform extraction as described by Premachandra et al. [30]. Leaf samples were immersed in $2 0 ~ \mathrm { m L }$ of chloroform in a Petri dish of $9 0 \ \mathrm { m m }$ diameter for 5  s. Te solvent was evaporated in a fume hood under a dry air stream, and the residue was allowed to dry for $2 4 \mathrm { ~ h ~ }$ at room temperature. After drying, the content of cuticular waxes was weighed by using a FA1004C electronic analytical balance and expressed on the basis of FW (fresh weight). 

# Regulation of gene expression by RT‑PCR

Te isolation of total RNA, the synthesis of cDNA and RT-PCR analysis were conducted according to our previous study [31]. Primers for FRO2, Nramp3, ABCG12 and WIN1 genes were designed based on the sequences available in NCBI genbank using the PrimerQuest (Integrated DNA Technologies, Coralville, IA) as described in Table 1. 

# Statistical analysis

Each treatment was conducted with three replicates, and the results were presented as mean $\pm \mathrm { \ S D }$ (standard deviation). Te statistical analysis of experimental data was verifed with the one-way ANOVA followed by Duncan’s multiple comparison $( p < 0 . 0 5 )$ in the statistical package IBM SPSS Version 22.

# Iron distribution and iron‑related gene expression in C. maxima plants

Te possible pattern of transformation and uptake of iron in C. maxima leaves is shown in Fig. 3A. Unfortunately, iron regulated transporter (IRT1) gene in citrus has not been sequenced yet. As expected, Fe concentration of $C _ { }$ . maxima shoots exposed to both $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs and $\mathrm { F e } ^ { 3 + }$ treatment increased rapidly with the increase of applied dosages (Fig. 3B). After exposure to 20, 50 and $1 0 0 ~ \mathrm { { m g / L } }$ $\gamma { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 } \mathrm { N P s }$ , Fe content in shoots was increased by 1.34, 3.78 and 6.77 times, respectively, relative to the control plants. Fe level of $\mathrm { F e } ^ { 3 + }$ treatments was elevated by 2.33, 4.38, 8.62 times, respectively. In addition, the total Fe content in C. maxima shoots was not signifcantly diferent between Fe(II)-EDTA and control plants. In C. maxima roots, no obvious diference of Fe levels was noted between all Fe treatments and control plants. 

FRO2 gene encodes a ferric chelate reductase, which can be activated when plants lack available Fe. As seen in Fig. 3C, the relative FRO2 gene expression of control was at a high level. $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs and $\mathrm { F e } ^ { 3 + }$ treatments led to $2 9 . 4 \mathrm { - } 9 1 . 4 \%$ lower levels of FRO2 gene expression than that of untreated control plants. Especially, $5 0 ~ \mathrm { m g / L }$ $\mathrm { F e } ^ { 3 + }$ treatment signifcantly decreased FRO2 expression to a much lower level than other treatments. # Discussion

Growth and physiological efects of γ‑Fe O NPs and $\mathsf { F e } ^ { 3 + }$ Fe(II)-EDTA, as one of the most widely used supplements for improving Fe availability to plants [35], showed no evident promotion to plant growth via foliar application 

in our study. Meantime, $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs did not exhibit any superiority in overcoming Fe defciency-induced chlorosis. We did not observe any evident diference of chlorophyll levels among treatments of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs, $\mathrm { F e } ^ { 3 + }$ , control and Fe(II)-EDTA, although iron content in C. maxima shoots of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs and $\mathrm { F e } ^ { 3 + }$ treatments was higher than control and Fe(II)-EDTA treatment. It is possible that iron was mainly used in other physiological reactions and thus no signifcant changes in chlorophyll content were observed. It is noteworthy that previously we found that through root exposure in a hydroponic system, $\mathrm { 0 { - } 1 0 0 \ m g / L }$ of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs and $\mathrm { F e } ^ { 3 + }$ had a dosedependent efect on chlorophyll synthesis of C. maxima [31]. $5 0 ~ \mathrm { m g / L }$ $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs and all $\mathrm { F e } ^ { 3 + }$ treatments notably increased chlorophyll levels. Tey used diferent parameters, including $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 } \mathrm { N P }$ size and concentrations, growth condition, treatment time as well as plant species, which may explain the contradictory results from ours. 

To demonstrate if $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs altered the plant health at physiological level, we analyzed the change of soluble proteins, which is an important indicator of plants’ defense. Plants could adapt themselves to various stresses by producing soluble proteins as osmolytes [36], antioxidants, or scavengers for eliminating free radicals in plants [37]. For example, Afaq et  al. [38] observed an increase in the antioxidant enzymes after $\mathrm { T i O } _ { 2 }$ NPs treatment as indicated at the transcriptional or protein level. Meanwhile, it is known that various abiotic stresses lead to the overproduction of ROS in plants which are highly reactive and toxic, ultimately resulting in oxidative stress and protein damage [39]. Nevertheless, in this study, no oxidative stress was induced in plants exposed to $2 0 ~ \mathrm { m g / L }$ $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs, based on the results of MDA content and the antioxidant enzyme activities (Fig. 2B–E), indicating 

that the lower soluble protein level could be caused by an alternative mechanism, instead of protein damage caused by overproduction of ROS. Meantime, the unchanged soluble protein contents under other treatments might be a result of self-regulation by plants. 

Oxidative stress caused by $\mathsf { v } { \mathsf { - F e } } _ { 2 } \mathsf { O } _ { 3 } \mathsf { N P s }$ and $\mathsf { F e } ^ { 3 + }$ on plants In this study, no elevated MDA level in shoots under $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs exposure was induced, suggesting that either foliar applied $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs did not induce lipid peroxidation even at high exposure concentrations or the plant’s detoxifcation pathways were sufcient to address and remedy the induced stress [40]. Activities of three antioxidant enzymes in plants treated with 20 and $5 0 ~ \mathrm { \ m g / L }$ $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs were unafected, while $1 0 0 ~ \mathrm { { m g / L } }$ $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs signifcantly increased the activity of CAT and POD. Higher activity of CAT and POD can contribute to the detoxifcation of excessive amounts of $\mathrm { H } _ { 2 } \mathrm { O } _ { 2 }$ [41]. Given the results of MDA levels and antioxidant enzymes, it is clear that 20 and 50 mg/L $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs did not induce oxidative stress in plant shoots, while 100 mg/L γ-Fe O NPs might initially cause ROS generation but then the plant’s defense systems remedied the induced stress. As for $\mathrm { F e } ^ { 3 + }$ treatments, 20 and $1 0 0 ~ \mathrm { { m g / L } }$ treated shoots showed a much higher MDA content, while that of $5 0 \mathrm { \ m g / L \ F e ^ { 3 + } }$ treatment was unafected compared with the control. However, no elevated activities of three antioxidant enzymes under $2 0 \mathrm { \ m g / L \ F e ^ { 3 + } }$ $\mathrm { F e } ^ { 3 + }$ 

treatment were observed, indicating that the increase of MDA level under $2 0 ~ \mathrm { m g / L ~ F e ^ { 3 + } }$ was an abnormal result. Combined MDA content with the higher POD activity of 50 and $1 0 0 ~ \mathrm { m g / L ~ F e ^ { 3 + } }$ treatments in shoots, it could be deduced that C. maxima treated with $5 0 \mathrm { m g / L } \mathrm { F e } ^ { 3 + }$ could address and remedy the induced oxidative stress, while plants treated with $1 0 0 \mathrm { \ m g / L \ F e ^ { 3 + } }$ were not sufcient to deal with stress induced by $\mathrm { F e } ^ { 3 + }$ at a high concentration. Te unchanged MDA production and antioxidant enzyme activities in C. maxima roots among all the treatments indicated that no oxidative stress occurred in plant roots. 

# Uptake and translocation of $\mathsf { \pmb { \gamma } } \mathsf { \pmb { \mathrm { F } } } \mathsf { e } _ { 2 } \mathsf {  { O _ { 3 } } }$ NPs

Iron content of C. maxima shoots exposed to diferent concentrations of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs showed a dose-dependent trend. Te higher Fe level of $\gamma { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 } \mathrm { N P s }$ in shoots indicated that signifcant uptake had occurred. Several studies demonstrated that $\mathrm { F e } _ { 2 } \mathrm { O } _ { 3 }$ NPs in a hydroponic system could enter plants through roots [42, 43], or silica sediment [10]. However, to our knowledge, few studies investigated whether foliar applied $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs could enter plant leaves and further translocate to roots or not. We observed the uptake of iron into shoots but no diference of iron content in C. maxima roots between all treatments, suggesting that no downward transport of iron occurred in C. maxima plants. In our previous study, we observed that root-applied $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs had no translocation from roots to shoots [31]. Terefore, either foliar spray or root supply of $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs alone cannot meet the requirement of the whole plants. A combination of both application methods may improve the efectiveness of iron fertilization in agricultural and horticultural production. 

Generally, when plants are deprived of iron, the new leaves become chlorotic and young lateral roots show the characteristic Fe-defcient stress-response mechanisms: enhanced Fe(III) reducing capacity, subapical swelling and acidifcation of the medium [44]. However, previous studies showed that leaf mesophyll cells also display plasma membrane ferric reductase activity [44, 45]. When a plant sufers from iron shortage, the reductive system is strongly activated [45], with FRO2 gene encoding a ferric chelate reductase. Te down-regulation of FRO2 gene expression under $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs and $\mathrm { F e } ^ { 3 + }$ treatments compared to the control and Fe(II)-EDTA treatment indicated that C. maxima could utilize the supplied iron in $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs and $\mathrm { F e } ^ { 3 + }$ via foliar application. Te relative level of FRO2 expression exposed to $5 0 ~ \mathrm { m g / L }$ $\mathrm { F e } ^ { 3 + }$ was the lowest. Te supply of iron is not only dependent on applied dosage, but also plants’ utilization ability. Based on the MDA data, 20 and $1 0 0 ~ \mathrm { m g / L ~ F e ^ { 3 + } }$ treatments had higher MDA formations than $5 0 ~ \mathrm { m g / L }$ 

$\mathrm { F e } ^ { 3 + }$ , which indicates that $5 0 ~ \mathrm { m g / L }$ leads to less oxidative stress than the other two dosages. Terefore, plants could better utilize $\mathrm { F e } ^ { 3 + }$ at $5 0 ~ \mathrm { m g / L }$ , which explains why the activation of FRO2 gene of $5 0 ~ \mathrm { m g / L }$ was lower than 20 and $1 0 0 ~ \mathrm { m g / L ~ F e ^ { 3 + } }$ treatments. Taken together, $5 0 ~ \mathrm { m g / L }$ $\mathrm { F e } ^ { 3 + }$ can supply higher amount of iron than $2 0 ~ \mathrm { m g / L }$ $\mathrm { F e } ^ { 3 + }$ . Meanwhile, the toxicities of $2 0 ~ \mathrm { { m g / L } }$ and $1 0 0 ~ \mathrm { { m g / L } }$ $\mathrm { F e } ^ { 3 + }$ are higher than that of $5 0 ~ \mathrm { m g / L }$ and likely inhibits the leaf ability to absorb and utilize iron. Also, $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs and $\mathrm { F e } ^ { 3 + }$ had a higher ability to supply iron to plants than Fe(II)-EDTA, except for $1 0 0 \mathrm { \ m g / L \ F e ^ { 3 + } }$ . In addition, the lower level of Nramp3 gene expression at all $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs concentrations indicated that plant was in iron-sufcient status. Te much higher level of Nramp3 gene expression of $1 0 0 \mathrm { m g / L } \mathrm { F e } ^ { 3 + }$ and Fe(II)-EDTA treatment than the control suggested that Fe(II)-EDTA and $\mathrm { F e } ^ { 3 + }$ at high concentrations cannot alleviate iron defciency via foliar spray. It was reported that Fe-chelates are more efective in soil than in foliar applications, and foliar Fe chelate-fertilization cannot yet be considered as a reliable strategy to control plant Fe-defciency [46]. Previous study showed that $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs is a suitable adsorbent for efectively extracting pollutants from the environment due to their high specifc surface area and accessible surface adsorption sites, which make them well applicable for the adsorption of pollutants [47, 48]. Given this, the strong adsorption ability of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs contributed to their stable attachment on the leaf surface and further absorption by plants. In agricultural production, most of the applied fertilizers are frequently lost due to the degradation by photolysis, leaching, hydrolysis, and decomposition [49]. It is essential to reduce nutrient losses in fertilization and increase the crop yield through the development of nanomaterials-based fertilizers [49]. In this regard, our results revealed that $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs have the potential to be an efective nanofertilizer and reduce nutrient loss during and after application. 

# Interaction between $\mathsf { v } { \mathsf { - F e } } _ { 2 } \mathsf { O } _ { 3 } \mathsf { N P s }$ and cuticular wax

In this study, iron distribution indicated that γ-Fe O NPs may be tightly attached to the leaf surface and/or taken up by the C. maxima leaves. Cuticular wax is a protective barrier on leaf epidermis, which could adsorb and trap intrusive NPs. Once NPs translocate through the cuticle, NPs could difuse in the leaf tissue. We observed a signifcantly lower expression levels of WIN1 gene under all Fe exposures. No upregulation of ABCG12 gene expression treated with $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs and $\mathrm { F e } ^ { 3 + }$ treatments was observed as well. However, wax contents of $1 0 0 ~ \mathrm { { m g / L } }$ $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs and $2 0 { - } 1 0 0 ~ \mathrm { \ m g / L } ~ \mathrm { F e } ^ { 3 + }$ treatment were signifcantly enhanced. Such an increase of wax content could hinder the uptake of high levels of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs and ionized iron $( \mathrm { F e } ^ { 3 + } )$ . Wax content is closely 

correlated with stress resistance of plants [50]. According to Fig. 2B–E, $\mathrm { F e } ^ { 3 + }$ treatments induced stress in plant shoots. Te higher wax levels of plant leaves under $\mathrm { F e } ^ { 3 + }$ treatments might be a result of anti-stress. Te high Fe content in shoots of $1 0 0 ~ \mathrm { { m g / L } }$ $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs suggested that most NPs were trapped on the surface wax as a result of the formation of clusters and large agglomerates [18]. Strangely, Fe(II)-EDTA treatment had a lower WIN1 gene expression level but a much higher ABCG12 gene expression level, while wax content of Fe(II)-EDTA treatment was at a notably high level. Fernández et  al. [46] reported that sprayed Fe-chelates could be taken up via the cuticle due to the comparable sizes of Fe-compounds and the pores. Te signifcantly higher wax content of Fe(II)-EDTA might be a mechanism of defending against alien substances. In addition to WIN1, there are many genes involved in the synthesis and secretion of surface wax [51]. For instance, CUT1, an Arabidopsis gene required for cuticular wax production, encodes a VLCFA condensing enzyme [33]. Terefore, the biosynthesis of wax is a collaborative and complicated process, which explain why $1 0 0 ~ \mathrm { { m g / L } }$ $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs and $\mathrm { 2 0 { - } 1 0 0 ~ m g / L }$ $\mathrm { F e ^ { 3 + } }$ led to higher wax content without inducing higher expression levels of WIN1 and ABCG12, as well as Fe(II)- EDTA treatment had the lower expression of WIN1 but higher level of wax. Moreover, Jetter et al. [52] reported that cuticular wax is typically a complex mixture of dozens of compounds with diverse hydrocarbon chain or ring structures. How much each of the wax compounds contributes to the overall biological functions of the cuticular wax is largely unknown [53]. Terefore, further explorations should be made to fgure out the processes and mechanisms underlying the interactions between NPs and cuticular waxes. 

# Conclusions

Based on the growth and physiological parameters, it is clear that foliar sprayed $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs and $\mathrm { F e } ^ { 3 + }$ at the concentrations used in this study had an inconsequential efect on plant growth as shown in chlorophyll content, fresh weight, and root activity. However, the expression of genes associated with the absorption and transformation of iron in leaves showed that plants were in iron-suffcient status. Further analysis of iron content shows no downward transport of iron from shoots to roots in all treated forms via foliar application. It is well known that iron is hard to transport from leaves. As for lipid peroxidation, all $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs exposures showed insignifcant changes as compared with the control. Antioxidant analysis indicated that 20 and $5 0 ~ \mathrm { m g / L }$ $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs induced no oxidative stress while $1 0 0 ~ \mathrm { { m g / L } }$ $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs may induced stress initially but plants were sufcient to deal with it. Moreover, the higher wax content of $1 0 0 ~ \mathrm { { m g / L } }$ 

$\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs as compared with the control would hinder the uptake of high levels of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs. Results of WIN1 and ABCG12 gene expression revealed that the biosynthesis of wax is a collaborative and complicated process and more than one gene are involved in this process. Commendably, foliar applied $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs have the ability to reduce nutrient loss probably due to the strong adsorption ability and gradual Fe release. Given that no phytotoxicity of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs at lower concentrations (20 and $5 0 ~ \mathrm { m g / L }$ ) was observed, it is possible that using $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs at lower doses is feasible to enhance the utilization and efciency of inorganic iron fertilizer in agricultural production. Moreover, in real applications, foliar sprayed $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs may be utilized together with soil supplied $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs to alleviate chlorosis and improve the iron use efciency. Our fndings provide a novel perspective to the interactions between foliar-applied NPs and plants, and will inspire further critical eforts to systemically explore the potential applications of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs in agronomic production. 

Tere is still much unknown about the speciation change of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs during plant foliar interactions. Further eforts should be made to determine (1) if the $\mathsf { Y } { \mathrm { - F e _ { 2 } O _ { 3 } } }$ NPs are absorbed as NPs directly, or dissolution occurs inside or outside plant leaves with free iron ions available for uses by plant leaves; (2) if $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs pass through leaf epidermis as NPs, what is their fnal speciation after interacting with leaf organelles? In addition, $2 0 ~ \mathrm { m g / L }$ may not be the lowest concentration to supply sufcient iron for plants. Concentrations of $\mathsf { Y } { \mathrm { - F e } } _ { 2 } \mathrm { O } _ { 3 }$ NPs lower than $2 0 \mathrm { m g / L }$ should be tested in the future.
