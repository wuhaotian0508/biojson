# Abstract

The transfer of one-carbon $( \mathsf { C } _ { 1 } )$ units is an integral part of cellular metabolism and is essential for the biosynthesis of nucleotides, amino acids, and cofactors, as well as for cellular methylation reactions. Within the plant cell, mitochondria are considered the hubs of one-carbon metabolism; however, the mechanisms and fluxes that distribute $\mathrm { C } _ { 1 }$ units from the mitochondria throughout the cell are unknown. Formate, the anion of formic acid, is an intermediate of $\mathrm { C } _ { 1 }$ metabolism and is converted to $\mathrm { C _ { 1 } }$ -tetrahydrofolate intermediates $\mathrm { \Delta C _ { 1 } }$ folates) or oxidized to $\mathrm { C O } _ { 2 }$ by formate dehydrogenase. The presence of formate dehydrogenase in plant cells challenges the formate exchange between mitochondria and the cytosol, a basic principle of eukaryotic cellular and organellar $\mathrm { C } _ { 1 }$ metabolism. Based on the biochemical and physiological characterization of Arabidopsis (Arabidopsis thaliana) formate dehydrogenase 1 (FDH1), we propose an FDH1-regulated $\mathrm { C _ { 1 } }$ shunt linking mitochondrial and cytosolic $\mathrm { C _ { 1 } }$ metabolism by formate exchange. Finally, we give a perspective on a cellular serine/ formate shuttle that allows the distribution and transfer of $\mathrm { C _ { 1 } }$ units according to the redox state within the compartments.

# Introduction

One carbon $\left( \mathsf { C } _ { 1 } \right)$ metabolism is essential to all living organisms (Hanson et al. 2000). In plants, folate (vitamin $\mathrm { B } _ { 9 , }$ ) co-enzymes, supply activated $\mathrm { C _ { 1 } }$ units for a variety of metabolic processes. The biosynthesis of nucleotides (purine and thymidylate), mitochondrial and chloroplastic proteins, vitamin as well as amino acid metabolism and methylation reactions all depend on the provision of $\mathrm { C } _ { 1 }$ units (Hanson and Roje 2001). Within the cell, tetrahydrofolate (THF) functions as carrier for $\mathrm { C _ { 1 } }$ units via $\mathrm { C } _ { 1 }$ THF $( \mathsf { C } _ { 1 }$ folates (Hanson and Roje 2001). $\mathrm { C _ { 1 } }$ folates exist in several oxidation states (10-formyl-THF as most oxidized and 5-methyl-THF as most reduced form) and are reversibly converted between the different oxidation states (Hanson and Gregory 2011).

Understanding of the mechanisms distributing folate-derived $\mathrm { C } _ { 1 }$ units among subcellular compartments in plants is limited. Plant folate transporters have been documented, but their importance in the $\mathrm { C _ { 1 } }$ unit distribution is unclear. The AtFOLT1 transporter has been reported to function in chloroplast uptake of folates (Bedhomme et al. 2005) but was tested for transport capability only with folic acid. Thus, it is not clear if it has the ability to transport the $\mathrm { C } _ { 1 }$ folate species. The At2g32040 protein has been shown to transport the less abundant monoglutamlylated, but not polyglutamylated, 5-formyl-THF, into chloroplasts

(Klaus et al. 2005; Eudes et al. 2010). One vacuolar transporter has been identified and shown to transport a variety of folate species (Raichaudhuri et al. 2009). Although transport of tetrahydrofolate out of mitochondria, which catalyze the final steps of folate biosynthesis, must exist to satisfy the needs for folates in other subcellular compartments, it is not yet known whether $\mathrm { C } _ { 1 }$ folates are transported into and out of these organelles. In addition to the membrane transport, various other mechanisms likely exist to distribute $\mathrm { C } _ { 1 }$ units between compartments (Cybulski and Fisher 1981). Heterotrophic eukaryotes use a $\mathrm { C _ { 1 } }$ shunt based on formate exchange to link mitochondrial and cytosolic $\mathrm { C } _ { 1 }$ metabolism (Christensen and MacKenzie 2006; Ducker et al. 2016; Zheng et al. 2018). In mammalian cells, mitochondrial-derived formate fuels cytosolic $\mathrm { C } _ { 1 }$ metabolism for 10-formyl-THF formation, thereby maintaining cellular redox homeostasis and folate integrity (Ducker et al. 2016; Zheng et al. 2018). In plants, however, a similar $\mathrm { C } _ { 1 }$ shunt between mitochondria and the cytosol is challenged by the existence of a mitochondrial formate dehydrogenase (Li et al. 2000). In leaf mitochondria, photorespiration and $\mathrm { C } _ { 1 }$ metabolism interact through the conversion of glycine to serine, catalyzed by the combined activity of the glycine decarboxylase complex (GDC) and serine hydroxymethyltransferase (SHM). Both enzymes catalyze THF-dependent reactions and are major

soluble proteins of leaf mitochondria (Eisenhut et al. 2019; Busch 2020; Fuchs et al. 2020; Wittmiß et al. 2020; Kuhnert et al. 2021). The lethal phenotypes of the gldp1gldp2 (GDC P-protein), gldt1 (GDC T-protein), and shm1shm2 mutants in Arabidopsis thaliana (Arabidopsis) are explained by the dual role of GDC and SHM in $\mathrm { C _ { 1 } }$ metabolism and define mitochondria as the cellular hub for the provision of $\mathrm { C } _ { 1 }$ units within the plant cell (Engel et al. 2007, 2011; Timm et al. 2017). The central $\mathrm { C } _ { 1 }$ intermediate of the GDC/ SHM catalyzed reactions is 5,10-methylene-THF, derived from either glycine oxidation or reversible hydroxymethyl group transfer from serine. Besides glycine/serine metabolism, 5,10-methylene-THF is also oxidized in the mitochondrial THF cycle (Eisenhut et al. 2019; Busch 2020). The final step of the cycle, catalyzed by 10-formyl-deformylase (FDF), releases THF and produces a single formate molecule (Collakova et al. 2008; Eisenhut et al. 2019). The photorespiratory phenotype and progeny lethality of the 10-formyl-deformylase double knockout mutant provide additional evidence for the role of mitochondrial $\mathrm { C _ { 1 } }$ metabolism in photorespiration and plant development (Collakova et al. 2008). Furthermore, the THF cycle contributes to the cellular formate pool together with non-enzymatic glyoxylate decarboxylation, methanol metabolism, and oxalate catabolism (Igamberdiev et al. 1999; Wingler et al. 1999; Foster et al. 2012). Other contributors to the in planta formate pool include detoxification of formaldehyde by GSH (Achkor 2003; Song et al. 2013) and the methionine salvage cycle (Vigani et al. 2017 and references therein).

There are at least 2 metabolic pathways that determine the fate of formate, implying either a role as an intermediate in folatedependent $\mathrm { C _ { 1 } }$ metabolism or the oxidation to $\mathrm { C O } _ { 2 }$ (Hanson and Roje 2001). Biochemical data obtained by subcellular fractionation indicated that plants possess the formate-tetrahydrofolate ligase (FTHFL) enzymes, which catalyze the THF-dependent conversion of formate to 10-formyl-THF in plastids, mitochondria and the cytosol (Hanson and Roje 2001). However, expression of the N- or C-terminally GFP-tagged enzyme from Arabidopsis (AtFTHFL) revealed a signal only in the cytoplasm, indicating that AtFTHFL is a cytosolic enzyme (Saeheng et al. 2024). In the cytosol, 10-formyl-THF can be converted to more reduced $\mathrm { C } _ { 1 }$ -folate species, which are then used for thymidylate and methionine biosynthesis (Groth et al. 2016; Gorelova et al. 2017). Methionine is an integral part of the methyl cycle to produce S-adenosyl-Lmethionine (SAM), the cellular methyl group donor (Ranocha et al. 2001; Sauter et al. 2013). Although plastids are autonomous for methionine biosynthesis, they are dependent on the import of SAM as a methyl donor by the SAMT1 transporter (Ravanel et al. 2004; Bouvier et al. 2006). The importance of cytosolic $\mathrm { C } _ { 1 }$ metabolism was demonstrated by characterizing the hypomorphic and dwarfed mutant of the cytosolic methylenetetrahydrofolate dehydrogenase/methenyltetrahydrofolate cyclohydrolase activity (MTHFD1). Loss of MTHFD1 activity abolishes the SAM cycle and causes genome-wide DNA hypomethylation and loss of histone H3K9 methylation (Groth et al. 2016). Due to the presence of mitochondrial NAD-dependent formate dehydrogenase (FDH), formate could also be oxidized to $\mathrm { C O } _ { 2 }$ while reducing a pyridine cofactor, preferentially NADH. The existence of FDH has been demonstrated in several plant species (potato, rice, Arabidopsis) and its role as a stress-inducible protein has been hypothesized (Colas des Francs-Small et al. 1993; Hourton-Cabassa et al. 1998; Li et al. 2000; Shiraishi et al. 2000; Alekseeva et al. 2011; Choi et al. 2014; Lou et al. 2016). Furthermore, FDH is regulated by post-translational modifications, including phosphorylation and acetylation, but the consequences for FDH activity remain unknown (Bykova et al. 2003; Kuhnert et al. 2020;

Møller et al. 2020). In Arabidopsis, FDH is encoded by a single gene (hereafter referred to as FDH1), and several studies at the beginning of the century reported on the biochemical characterization of FDH1 and the dual targeting of the enzyme to mitochondria and chloroplasts (Li et al. 2000, 2002; Olson 2000; Herman et al. 2002; Baack et al. 2003).

Here, we address the limited understanding of $\mathrm { C _ { 1 } }$ metabolism in plants by characterization of Arabidopsis FDH1. We propose that plants use a $\mathrm { C _ { 1 } }$ shunt based on formate exchange to link mitochondrial and cytosolic $\mathrm { C _ { 1 } }$ metabolism and that FDH1 regulates flux through the shunt. To this end, we analyzed the kinetic constants of recombinant FDH1 and defined the enzyme as NAD-dependent FDH. In addition, we confirmed that FDH1 is localized to plant mitochondria and is not dual-targeted to the chloroplast as previously described (Li et al. 2000, 2002; Olson 2000; Baack et al. 2003). We also investigated the role of FDH1 in planta by generating loss-of function mutants and overexpression lines. Loss of FDH1 reduced plant growth, altered chlorophyll biosynthesis, and reduced photosynthesis, whereas overexpression increased plant growth. Finally, we proposed a potential serine/ formate shuttle to mediate the transfer of $\mathrm { C } _ { 1 }$ units between compartments according to the cellular and/or organellar redox state.

# Results

# FDH1 is a NAD-dependent formate dehydrogenase

Biochemical characterization of FDH1 has reported kinetic constants that differ by an order of magnitude with respect to the affinity for formate, ranging from 1.4 to $1 0 \mathrm { m M }$ (Li et al. 2000) (Olson 2000). To accurately determine the kinetic constants of FDH1, we heterologously expressed FDH1 in E. coli and purified the protein by immobilized metal affinity chromatography using an N-terminal 6xHis-tag. Recombinant FDH1 had a $K _ { \mathrm { m } }$ value of 3.85 $\pm 0 . 1 4 \mathrm { m M }$ for formate and a $\mathtt { V } _ { \mathrm { m a x } }$ of $1 . 0 2 \pm 0 . 0 1$ µmol min−1 mg−1 protein as determined by Michaelis–Menten kinetics and showed no activity with NADP (Supplementary Figure S1). In a recently published mitochondrial proteomic dataset, we identified the first 30 N-terminal amino acids of FDH1 as mitochondrial target peptide that is cleaved upon mitochondrial import (Kuhnert et al. 2020). Comparison of FDH1 activity with a variant lacking the mitochondrial target peptide $( \mathrm { F D H 1 } ^ { \Delta 1 - 3 0 } )$ ), revealed no significant differences in total FDH activity (Fig. 1a), although both, the $K _ { \mathrm { m } }$ value for formate $( 4 . 1 1 \pm 0 . 1 \mathrm { m M } )$ and $\mathrm { V } _ { \mathrm { m a x } } \left( 1 . 4 7 \pm 0 . 0 2 \mu \mathrm { m o l \ m i n } ^ { - 1 } \mathrm { m g } ^ { - 1 } \right.$ $\mathrm { V } _ { \mathrm { m a x } } \left( 1 . 4 7 \pm 0 . 0 2 \right.$ protein) slightly increased (Supplementary Figure S1).

We considered only FDH1 for further biochemical characterization and determined a pH optimum of 7.5 (Fig. 1b). Formate was the preferred substrate of FDH1, as no activity was detected with glutamate, glycolate, acetate, or oxalate (Fig. 1c). However, glyoxylate was accepted as a secondary substrate with $4 \%$ activity compared to formate (Fig. 1c). Furthermore, FDH1 was not allosterically regulated by photorespiratory metabolites (glycine, serine, glycolate, glyoxylate), glutamate and oxalate (Fig. 1d); nor did oxidizing or reducing conditions alter the activity (Fig. 1e and f).

# Absence or overexpression of FDH1 alters photosynthesis and respiration

Given the observed growth phenotype, we evaluated photosynthesis in all genotypes by gas-exchange measurements. Therefore, $\mathrm { A / C _ { \mathrm { i } } }$ curves, the rate of $\mathrm { C O } _ { 2 }$ assimilation (A) in relation to intercellular

$\mathrm { C O } _ { 2 }$ concentration (Ci) under saturating light (1,000 µmol $\mathsf { s } ^ { - 1 } \mathsf { m } _ { 2 } ^ { - 2 } )$ and light curves (assimilation at constant $\mathrm { C O } _ { 2 }$ under sub saturating light) were generated using mature rosette leaves. From the $\mathsf { A } / \mathsf { C } _ { \mathrm { i } }$ curves, all 3 fdh1 mutants were reduced in their maximal assimilation rates (Table 1). Furthermore, in 2 of the 3 mutants, the $\mathrm { C O } _ { 2 }$ compensation point was elevated and the slope of the $\mathrm { A / C _ { \mathrm { i } } }$ curve was reduced (for fdh1-2 $P { = } 0 . 0 5 4$ , Table 1). In OEX:FDH1#3 plants the maximum assimilation rate was significantly increased and the slope of the A/C curve was increased in OEX:FDH#2 and #3 (Table 1). We further modeled the $\mathrm { A / C _ { \mathrm { i } } }$ curve using the Farquharvon Caemmerer—Berry model of $C _ { 3 }$ photosynthesis (Farquhar et al. 1980). Based on the model, we deduced that $\mathrm { V } _ { \mathrm { c m a x } }$ , the maximum rate of ribulose 1,5-bisphosphate carboxylation and Jmax, the maximum rate of electron transport were reduced in the fdh1 mutants. In OEX:FDH1#2 and #3 plants, $\mathtt { V } _ { \mathrm { c m a x } }$ was increased whereas $J _ { \mathrm { m a x } }$ was not altered in FDH1 overexpression plants. Both the light saturated rate of photosynthesis and the light compensation point were not altered between the genotypes. However, dark respiration was reduced in the fdh1-2 and fdh1-3 mutants and the quantum efficiency was reduced in the fdh1-1 and fdh1-3 mutants. While dark respiration was not altered in the overexpression plants, the quantum efficiency was slightly increased in OEX:FDH1#3 (Table 1).

# Discussion

The underlying fundamental principles of $\mathrm { C _ { 1 } }$ metabolism in plants are poorly defined due to the high genetic redundancy, low metabolic flux, metabolite instability, and mutant lethality (Hanson and Roje 2001). Formate, an intermediate of $\mathrm { C } _ { 1 }$ metabolism, is either utilized in folate-dependent $\mathrm { C _ { 1 } }$ metabolism or oxidized to $\mathrm { C O } _ { 2 }$ (Ducker and Rabinowitz 2017). In mammalian cells, formate exchange between mitochondria and the cytosol is essential for compartmentalized folate-dependent $\mathrm { C _ { 1 } }$ metabolism (Fan et al. 2014; Ducker et al. 2016; Zheng et al. 2018). However, a comparable $\mathrm { C _ { 1 } }$ shunt in plants is challenged by the existence of mitochondrial FDH, which was previously thought to be dual-localized to the

chloroplast (Olson 2000; Herman et al. 2002; Alekseeva et al. 2011). Based on the characterization of the role of Arabidopsis FDH1, we propose that in plants, FDH regulates the flux of the $\mathrm { C _ { 1 } }$ shunt that mediates formate exchange between mitochondria and the cytosol to maintain homeostatic $\mathrm { C _ { 1 } }$ metabolism.

In Arabidopsis, FDH is encoded by a single gene (FDH1) and localized to the mitochondria (Fig. 2, Supplementary Figure S2). Given the expected low intracellular formate levels in the plant cell, the high $K _ { \mathrm { m } }$ for formate of recombinant FDH1 was surprising (Wingler et al. 1999; Supplementary Figure S1). However, it implies either underestimated cellular formate levels or a small pool size, that is maintained below a toxicity threshold (Fig. 4, (Stemler 1980; Blubaugh and Govindjee 1988; Igamberdiev et al. 1999)). Furthermore, kinetic constants determined in vitro may differ from in vivo scenarios based on the cellular environment and regulation of the enzyme. Indeed, redox regulation of the plastidial glucose 6-phosphate dehydrogenase, the first enzyme of the glucose 6-phosphate shunt, increases the $K _ { \mathrm { m } }$ of the enzyme under reduced conditions (Scheibe et al. 1989; Hauschild and von Schaewen 2003; Sharkey and Weise 2016). Although FDH1 activity was not affected by oxidized or reduced conditions (Fig. 1), posttranslational modification, by phosphorylation and/or acetylation, could play a role in regulating FDH activity and kinetics (Bykova et al. 2003; Kuhnert et al. 2020; Møller et al. 2020). We note that FDH1 was identified in an Arabidopsis mitochondrial lysine acetylome study (König et al. 2014). However, a recombinant FDH1K184R acetylation mimic mutant, with the mutated lysine in the NAD-binding pocket showed no altered activity (Supplementary Figure S9) and kinetics suggesting that the acetylation site might be distant from the active site, as recently demonstrated for mitochondrial malate dehydrogenase (Balparda et al. 2022).

The photorespiratory phenotype of the 10-deformylase double knockout mutant indicates that 5,10-methylene-THF produced by the GDC is oxidized in the mitochondrial THF cycle and formate is produced (Collakova et al. 2008). This implies that in an illuminated leaf, the GDC is one source of mitochondrially derived formate. Consistent with the hypothesis of mitochondrial formate export via the shunt for cytosolic folate-dependent $\mathrm { C } _ { 1 }$ metabolism, the transcriptional abundance of FDH1 within a light phase behaved acyclic to GLDT (Supplementary Figure S10). The GLDT gene encodes the GDC T-protein, which is required for the production of 5,10-methylene-THF, which is oxidized by the THF-cycle to produce formate. However, a quantitative estimate of the fate and flux of photorespiratory $\mathrm { C _ { 1 } }$ metabolism is lacking (Busch 2020). Stable isotope tracing with $^ { 1 3 } \mathrm { C O } _ { 2 }$ and 33sulfur using the soluble methionine pool as a readout of $\mathrm { C _ { 1 } }$ fluxes, does not distinguish between cytosolic and/or plastidial SHM and formate-dependent folate metabolism, both of which contribute to the 5,10- methylene-THF pool used for methionine synthesis (Gauthier et al. 2010; Abadie and Tcherkez 2019).

To provide in planta evidence for the hypothesized $\mathrm { C } _ { 1 }$ shunt and the associated regulatory role of FDH1, we generated FDH1 knockout mutants and overexpression plants (Figs. 3 and  5, Supplementary Figure S4). The stunted growth of the fdh1 mutants, as reflected by reduced photosynthesis (Table 1), is likely due to impaired $\mathrm { C } _ { 1 }$ metabolism. Indeed, dysfunctional cytosolic $\mathrm { C _ { 1 } }$ metabolism in a hypomorphic and dwarfed mthfd1 mutant, caused DNA hypomethylation and reduced folate levels (Groth et al. 2016). In addition, chlorophyll biosynthesis depends on the integrity of the cel - lular folate pool. Depletion of the THF pool by methotrexate caused decreased chlorophyll biosynthesis, whereas the altered chloro - phyll a/b ratio in the fdh1 mutants could be a consequence of an altered folate pool (Fig. 5e; (Van Wilder et al. 2009). Remarkably, FDH1 overexpression accelerated plant growth (Fig. 5, Supplementary Figure S5). At least 3 effects could account for the observed phenotype. First, it was shown that overcoming the rate limiting steps of photorespiration by GDCH protein overexpression increases plant growth (Timm et al. 2012, 2015; Lopez-Calcagno et al. 2019). It remains to be demonstrated that higher mitochondrial FDH activity by FDH1 overexpression accelerates THF recycling and the GDC-catalyzed reaction of photorespiration, as glycine and serine steady state pools remained unchanged (Supplementary Figure S5). Furthermore, NADH produced by formate oxidation could be used in the redox shuttles and the respiratory chain to positively influence the cytosolic ATP/ADP ratio, as well as the NADH/ NAD ratio (Gardeström and Wigge 1988; Wigge et al. 1993; Igamberdiev and Gardeström 2003; Tomaz et al. 2010; Tcherkez et al. 2012). Finally, a positive effect of respired $\mathrm { C O } _ { 2 }$ on the Calvin-Benson-Bassham cycle seems plausible and could explain the increased maximum assimilation rate and the slope of the A/ $\mathrm { C _ { i } }$ curve in the strong FDH1 overexpression lines (Table 1).

However, all 3 explanations imply a plastic system to ensure the integrity of $\mathrm { C _ { 1 } }$ metabolism. Consistent with this, we did not observe changes in whole-cell pool sizes (glycine, serine, methionine, folates, formate, and SAM, Supplementary Figures S6–S8). Based on the comparison with mammalian $\mathrm { C } _ { 1 }$ metabolism and

the unlikely transport of $\mathrm { C } _ { 1 }$ folates across organellar membranes, we propose a formate/serine shuttle between the compartments in the plant cell (Cybulski and Fisher 1981; Ducker et al. 2016; Zheng et al. 2018; Li et al. 2020; Yang et al. 2020). Such a shuttle would allow the inter-compartment distribution of $\mathrm { C _ { 1 } }$ units and the adjustment of fluxes according to the redox state in each compartment. It should be noted that with the exception of the methylenetetrahydrofolate reductase (Roje et al. 1999), the specificity of the plant enzymes in $\mathrm { C _ { 1 } }$ metabolism for NAD or NADP remains unknown. Therefore, the following hypothesis is based on the specificity of mammalian counterparts and considers the ratio of reduced/oxidized NAD(P) in the compartments of the plant cell.

In an illuminated leaf, the combined activity of GDC and SHM1 in the context of photorespiration represents the primary pathway of serine biosynthesis (Prabhu et al. 1996, 1998; Voll et al. 2006; Igamberdiev and Kleczkowski 2018, Fu et al., 2023). However, GDC-derived 5,10-methylene-THF is also oxidized in the THF cycle to produce NADH, which is used by the respiratory chain or redox shuttles (Gardeström and Wigge 1988; Wigge et al. 1993; Douce et al. 2001; Tomaz et al. 2010; Selinski and Scheibe 2019; Shameer et al. 2019). The NADH dependence of the mammalian mitochondrial MTHFD and the slightly oxidized mitochondrial NAD pool support NADH production by $\mathrm { C _ { 1 } }$ metabolism (Christensen et al. 2005; Fan et al. 2014). Generated formate is assimilated in the cytosol and reduced to 5,10-methylene-THF under NADPH consumption, given a cytosolic NADPH/NAPD ratio of 1.5 (Gardeström and Wigge 1988; Igamberdiev and Gardeström 2003). This shuttle would complement the role of the cytosolic SHM providing $\mathrm { C _ { 1 } }$ units and could play a regulatory role in mediating the flux between SAM- and thymidylate biosynthesis (Herbig et al. 2002). Depending on the redox state, the fluxes could be reversed and formate produced in the cytosol could be oxidized in the mitochondria to produce NADH and $\mathrm { C O } _ { 2 }$ as needed.

In chloroplasts, 10-formyl-THF is required for purine biosynthesis and 5,10-methylene-THF is required for methionine biosynthesis. It has been proposed that these $\mathrm { C _ { 1 } }$ units are produced from formate (Hanson and Roje 2001; Ravanel et al. 2004; Zrenner et al. 2006), although no enzyme has yet been identified in chloroplasts that would direct formate into $\mathrm { C _ { 1 } }$ metabolism. Alternatively, $\mathrm { C } _ { 1 }$ units could be provided by plastidial SHM3. SHM3 has been biochemically characterized, but a mutant has not yet been reported (Zhang et al. 2010). In the absence of such data, it is difficult to draw firm conclusions about the role of plastidial SHM in providing the 5,10-methylen-THF pool. Analyses of a shm3 mutant would be required to fully understand plastidial $\mathrm { C _ { 1 } }$ metabolism and the genes involved.

In contrast to an illuminated leaf, the phosphoserine pathway is the dominant pathway of serine biosynthesis at night and in heterotrophic tissues (Benstein et al. 2013; Cascales-Miñana et al. 2013; Wulfert and Krueger 2018). In mitochondria, serine catabolism by SHM2 and the GDC produces 2 5,10-methylene-THF molecules per serine and one NADH. The subsequent complete oxidation of the 5,10-methylene-THF produces 4 additional NADH molecules, which may be particularly important under energylimited conditions to maintain respiration (Mouillon et al. 1999; Engel et al. 2011; Nunes-Nesi et al. 2014; Yang et al. 2020).

In general, the plasticity of the proposed formate/serine shuttle ensures the distribution of $\mathrm { C } _ { 1 }$ units between compartments. Furthermore, $\mathrm { C _ { 1 } }$ metabolic fluxes can be rapidly adjusted according to the cellular and organellar redox state. Metabolic flux analysis using formate- $^ { 1 3 } \mathrm { C }$ and serine- $^ { 1 3 } \mathrm { C }$ , in the fdh1 mutants and overexpression plants as well as mutants of shm1, shm2, and shm3 mutant would be required to fully resolve $\mathrm { C } _ { 1 }$ metabolic fluxes within the plant cell (Voll et al. 2006; Zhang et al. 2010; Engel et al. 2011). The lack of observed changes in the steady-state formate levels between the fdh1 mutant, the overexpression lines, and the wild type plants supports the formate/serine shuttle hypothesis. In fdh1, formate would be exported from mitochondria to the cytosol, where it would enter the cytosolic folate metabolism via the formatetetrahydrofolate ligase pathway. In the overexpressors, the transport might work the other way around. We take this as evidence for the plasticity of the cellular one-carbon metabolic network. One or the other enzyme (FDH or FTHFL) steps up as needed. Double mutants in FDH FTHFK will be needed to test this hypothesis.

We did not observe any changes in folate levels, indicating that de novo biosynthesis of the tetrahydrofolate cofactor is not affected. However, we cannot say whether there is a difference in the ratio of THF/5,10-methylene-THF. 5,10-methylene-THF is a critical metabolite because it is a branching point from which the $\mathrm { C _ { 1 } }$ units are directed to different uses: for (i) $\mathrm { C H } _ { 3 }$ -THF biosynthesis, which commits them to the use for the biosynthesis of Met, SAM, and the methylated products in the cytosol, or for (ii) serine biosynthesis if needed, or for (iii) dUMP to dTMP conversion and ketopantoate biosynthesis. Based on our data, we hypothesize that the steady-state levels do not change, but the flux does.

# Conclusion

Taken together, mitochondrial FDH1 appears to be critical for maintaining homeostatic $\mathrm { C } _ { 1 }$ metabolism by controlling flux across the $\mathrm { C _ { 1 } }$ shunt that connects the mitochondria to the cytosol (Fig. 6).

# Mitochondria isolation

Mitochondria were isolated from 4-wk-old Arabidopsis Col-0, fdh1-1, fdh1-2, and fdh1-3 rosette leaves via differential centrifugation and Percoll gradient purification, as described previously (Kühn et al. 2015). BSA was omitted from all solutions.

# Chlorophyll fluorescence and chlorophyll measurements

$\boldsymbol { F _ { \mathrm { v } } } / \boldsymbol { F _ { \mathrm { m } } }$ values of 12-d-old seedlings were measured using an imaging chlorophyll fluorometer (Imaging PAM, Walz, Effeltrich, Germany). Upon dark adaptation for 20 min, seedlings were exposed to a pulsed, blue probe beam and a saturating light flash to measure $\boldsymbol { F _ { \mathrm { v } } } / \boldsymbol { F _ { \mathrm { m } } }$ values. Chlorophyll was extracted from 4-wk-old leaves in $80 \%$ acetone and rotating at $4 ^ { \circ } C$ in the dark before photometric measurement. Chlorophyll content was calculated based on (Inskeep and Bloom 1985).

# Quantification of germination efficiency, primary root length, fresh weight and dry weight

Germination efficiency was quantified as seedlings establishment 12 d after transfer to light. For primary root length measurements, seeds were spotted on ½ MS plates and grown in vertical position for 12 d. Seedlings were photographed and primary root length was measured using Fiji (Schindelin et al. 2012). Fresh weight and dry weight were analyzed using 12-d-old seedlings and mature rosettes. Seedlings were collected in a 1.5 mL microcentrifuge tube and immediately weighted for fresh weight analysis. The same samples were dried at $6 5 ~ ^ { \circ } \mathrm { C }$ for 4 d and tubes were weighted again for dry weight analysis. Rosettes were harvested in bags and equally processed.

# Reverse transcription quantitative real time PCR

Total RNA from Arabidopsis tissue was extracted with Universal RNA kit (Roboklon). $2 \mu \mathrm { g }$ of total RNA was DNase treated with RNA-free DNase RQ1 (Promega). $1 \mu \mathrm { g }$ DNase-treated RNA was reverse-transcribed into cDNA using LunaScript RT SuperMix Kit (New England Biolabs). The reverse transcription quantitative real time PCR was carried out using Luna Universal qPCR Master Mix (New England Biolabs) and a StepOnePlus Real-Time PCR thermocycle (Applied Biosystems, Foster City, USA). PCR conditions were as followed: an initial denaturation step at $9 5 ~ ^ { \circ } \mathrm { C }$ for 10 min, followed by 45 cycles of $9 5 ~ ^ { \circ } \mathrm { C }$ for ${ 1 5 \mathrm { s } }$ , and $6 0 ~ ^ { \circ } \mathrm { C }$ for 60 s, followed by a melting curve. Gene-specific primers are listed in Supplementary Table S1. Transcript abundance was normalized to the reference gene Protein Phosphatase 2A Subunit A3 (PP2A, At1g13320, Czechowski et al. 2005). Relative transcript abundance was calculated as described by Simon (2003).

# Confocal laser scanning microscopy

Zeiss LSM780 confocal microscope and Zeiss ZEN software (Zeiss, Jena, Germany) was used for confocal laser scanning microscopy. In Arabidopsis, mitochondria were stained with the MitoTracker Green FM (ThermoFisher Scientific, Waltham, USA) and imaged according to the manufacturer’s instructions. Excitation/emission wavelengths were as followed: mCherry $( 5 6 1 \mathrm { n m } / 5 8 0$ to $6 2 5 \mathrm { n m }$ ), eGFP $( 4 8 8 \mathrm { n m } / 4 9 0$ to $5 5 0 \mathrm { n m }$ ), chlorophyll A $( 4 8 8 \mathrm { n m } / 6 4 0$ to $7 1 0 \mathrm { n m }$ ). Images were processed with Fiji (Schindelin et al. 2012).

# Gas exchange measurements

Mature rosette leaves of 6-wk-old plants grown at ambient $\mathrm { C O } _ { 2 }$ atmosphere were used for gas exchange measurements. Measurements were performed using a LICOR6800 (Licor Bioscience) with a flow set to $3 0 0 \mu \mathrm { m o l } \epsilon ^ { - 1 }$ $\mathsf { S } ^ { - 1 }$ , saturating light intensity of $1 , 0 0 0 \mu { \mathrm { m o l } } \mathrm { m } ^ { - 2 } \mathrm { s } ^ { - 1 }$ $\mathrm { m } ^ { - 2 } \thinspace \mathrm { s } ^ { - 1 }$ , leaf temperature of $2 5 ~ ^ { \circ } \mathrm { C }$ and a vapor pressure deficit below $1 . 5 \mathrm { k P a }$ . $\mathsf { A } / \mathsf { C } _ { \mathrm { i } }$ curves were measured via stepwise changes in external $\mathrm { C O } _ { 2 }$ supply ranging from 0 µbar to 1,600 µbar. Based on the $\mathrm { A / C _ { i } }$ curves, the maximum assimilation, the $\mathrm { C O } _ { 2 }$ compensation point as x-axis intercept and the slope of the $\mathrm { A / C _ { \mathrm { i } } }$ curve were determined. The $\mathsf { A } / \mathsf { C } _ { \mathrm { i } }$ curves were fitted with the classic Farquhar, von Caemmerer and Berry model of C3 photosynthesis (Farquhar et al. 1980) to estimate the maximum ribulose 1,5 bisphosphate saturated rate of carboxylation $\mathrm { ( V _ { c m a x } ) }$ , the maximum rate of electron transport $\left( J _ { \mathrm { m a x } } \right)$ and the $\mathrm { C O } _ { 2 }$ compensation points (CCP*). The light saturated rate of photosynthesis $( \boldsymbol { \mathrm { A } } _ { \mathrm { s a t } } )$ , the light compensation point (LCP, inferred as x-intercept), dark respiration $\mathrm { ( R _ { d } , }$ , photosynthesis rate without light), and the quantum efficiency (Qe, the slope of the light curve), were determined by light curve measurements at constant external $\mathrm { C O } _ { 2 }$ of 400 µbar and the light intensities were stepwise reduced from $1 , 0 0 0 \mu { \mathrm { m o l } } \mathrm { m } ^ { - 2 } \mathrm { s } ^ { - 1 }$ $\mathrm { m } ^ { - 2 } \thinspace \mathrm { s } ^ { - 1 }$ to $0 \mu \mathrm { m o l } \mathrm { m } ^ { - 2 } \mathsf { s } ^ { - 1 }$ .

# Metabolite profiling by GC-EI-QTOF-MS

For metabolite profiling green tissue of 14-d-old seedlings was harvested by liquid nitrogen dousing at the middle of the light phase. Frozen material was grinded using precooled mortar and pestil. Grinded material was aliquoted under continuous liquid nitrogen exposure to avoid sample thawing. Metabolites were extracted by one-phase extraction as previously described and normalized to ribitol as internal standard and the sample fresh weight (Fiehn et al. 2000).

# Folate analysis

Folate analysis was performed according to (Hung et al. 2013) with the following modifications. Approximately $1 0 0 \mathrm { m g }$ frozen sample was extracted using 1 mL folate extraction buffer (50 mM CHES-50 mM HEPES buffer [pH 7.9], 1 mM $\operatorname { C a C l } _ { 2 }$ , $2 \%$ (w/v) Na-ascorbate, $1 0 \mathrm { m M }$ $\beta$ -mercaptoethanol). Samples were incubated for $1 0 \mathrm { m i n }$ at $9 6 ~ ^ { \circ } \mathrm { C }$ and subsequently centrifuged for 10 min at 14,000 rpm and $4 ^ { \circ } C$ . The supernatant was collected, and the samples re-extracted with an additional 1 mL folate extraction buffer as described above. The supernatants were combined, and the extracts were deglutamylated by adding $1 0 0 \mu \mathrm { L }$ of dialyzed rat plasma. Samples were incubated at $3 7 ~ ^ { \circ } \mathrm { C }$ for $4 \mathrm { { h } }$ followed by boiling for 15 min to deactivate the enzymes. The denatured proteins were precipitated by centrifugation for 20 min at 14,000 rpm and $4 ~ ^ { \circ } \mathrm { C }$ . The supernatant was filtered through a $0 . 4 5 \mu \mathrm { m }$ PVDF membrane and subsequently applied to the folatebinding column, which was prepared as described previously (Konings 1999). Folates were eluted from the column by applying $0 . 5 \mathrm { m L }$ of HPLC mobile phase (Eluent B, 28 mM ${ \cal K } _ { 2 } \mathrm { H P O } _ { 4 }$ and $6 0 \mathrm { m M }$ $\mathrm { H } _ { 3 } \mathrm { P O } _ { 4 }$ [pH 2.5]) supplemented with $1 \%$ ascorbic acid. Folates were analyzed with the Alliance HPLC System (2,695 Separations Module, Waters, USA) with fluorescence detection (2,475 Multi-Wavelength Fluorescence Detector, Waters, USA) and the Cadenza CD-C18 column $3 \mu \mathrm { m }$ , $4 . 6 \times 2 5 0 \ : \mathrm { m m }$ , Imtakt Co., Japan) as described previously (Hung et al. 2013).

# S-adenosyl-L-methionine and formate analysis

S-adenosyl-L-methionine (SAM) analysis was performed as described previously (Hung et al. 2013) with the following modifications. Approximately $1 0 0 \mathrm { m g }$ frozen sample was extracted with 0.25 mL 0.1 M HCL. The samples were centrifuged for 20 min at 14,000 rpm and $4 ~ ^ { \circ } C$ . The supernatant was collected, the samples re-extracted as described above, and the supernatants were combined for further analysis.

For the SAM analysis, $4 9 \mu \mathrm { L }$ of 0.48 M citrate/0.59 M phosphate buffer $\mathrm { ( p H ~ 4 . 0 ) }$ , and $3 6 \mu \mathrm { L }$ of chloroacetaldehyde were added to $1 5 \mu \mathrm { L }$ of each sample. The samples were incubated at $8 0 ~ ^ { \circ } \mathrm { C }$ for 10 min (Rzewuski et al. 2007). The derivatized samples were immediately transferred to ice and subsequently centrifuged for 1 h at 14,000 rpm and $4 ~ ^ { \circ } \mathrm { C }$ . The samples were separated on the Xterra MS $C _ { 1 8 }$ column $5 \mu \mathrm { m }$ , $4 . 6 \times 1 0 0 \mathrm { m m }$ , Waters, USA) using Eluent A (TBAS buffer $\mathrm { \Delta } [ 5 . 7 \mathrm { m M }$ tetrabutylammoniumhydrogensulphate in 30.5 mM ${ \mathrm { K H } } _ { 2 } { \mathrm { P O } } _ { 4 }$ , pH 5.8]) and Eluent B (TBAS buffer and acetonitrile in a ratio of $1 { : } 2 \ [ \mathrm { v / v } ]$ ). The signal was detected at the excitation and emission wavelengths of $\mathfrak { X } _ { \mathrm { e x } }$ 280 and $\lambda _ { \mathrm { e m } }$ $4 1 0 \mathrm { n m }$ , respectively. SAM content in the samples was quantified using S- $5 ^ { \prime }$ -Adonosyl)-L-methionine chloride dihydrochloride as standard.

Formate analysis was performed as described previously (Xie et al. 2012) with the following modifications. $5 0 \mu \mathrm { L }$ of the extract was mixed with $5 0 ~ \mu \mathrm { L }$ of $2 0 0 \mathrm { m M }$ Tetrabutylammonium bromide in acetonitrile, $5 0 \mu \mathrm { L }$ of $2 . 5 \%$ triethanolamine in acetonitrile, ${ 1 \mu \mathrm { L } }$ of $1 0 \mathrm { m M }$ 9-chloromethyl anthracene as a fluorescencelabeling reagent, and $3 4 9 \mu \mathrm { L }$ of acetonitrile. The reaction was incubated at $7 5 ~ ^ { \circ } \mathrm { C }$ for 50 min. Samples were centrifuged for 10 min at 14,000 rpm and $2 5 ~ ^ { \circ } \mathrm { C }$ to remove debris. The samples were separated on the Xterra MS $C _ { 1 8 }$ column $( 3 . 5 \mu \mathrm { m }$ , $4 . 6 \times$ $1 0 0 \mathrm { m m }$ , Waters, USA) with a mobile phase of $6 4 \%$ acetonitrile and $3 6 \%$ water at the constant flow rate of $1 . 0 \mathrm { m L / m i n }$ at $3 0 ~ ^ { \circ } \mathrm { C }$ . The signal was detected at the excitation and emission wavelengths of $\texttt { \AA } _ { \mathrm { e x } } 3 6 5$ and $\mathtt { \AA } _ { \mathrm { e m } } 4 1 0 \mathrm { n m }$ , respectively. Formate

content in the samples was quantified using ammonium formate as standard.
