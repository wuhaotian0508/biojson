# N 6 -methyladenosine modifications stabilize phosphate starvation response–related mRNAs in plant adaptation to nutrientdeficient stress

$N ^ { 6 }$ -methyladenosine $( \mathsf { m } ^ { 6 } \mathsf { A } )$ , an abundant internal mRNA modification, is induced by various stress conditions and post-transcriptionally regulates gene expression. However, how $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications help plants respond to nutrientdeficiency stress remains unclear. Here, we profile high-confidence $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications in Arabidopsis transcriptome-wide under normal and inorganic orthophosphate (Pi)–deficient conditions (−P). High-confidence $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications are identified using synthetic modification-free RNA libraries for systematic calibration. Pi starvation induces widespread $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications, mediated by the Pi starvation response (PSR) master regulator PHOSPHATE STARVATION RESPONSE1 (PHR1) and its family members. Many Pi starvation–induced (PSI) $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications occur on PSR-related mRNAs, including PHR1. In addition, PHR1 proteins interact with the $\mathsf { m } ^ { 6 } \mathsf { A }$ writers MRNA ADENOSINE METHYLASE (MTA) and METHYLTRANSFERASE B (MTB) in nuclei under −P conditions. $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications facilitate systemic PSR signaling, as reflected by the reduced Pi content and PSR signaling in a knockdown artificial miRNA line targeting MTA, which shows a global decrease in $\mathsf { m } ^ { 6 } \mathsf { A }$ . 

Transcriptome-wide mRNA decay analysis reveals that PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ increases the stability of PSR-related mRNAs, but not through alternative polyadenylation site shifts. Analysis of transgenic plants with mutations in $\mathsf { m } ^ { 6 } \mathsf { A }$ loci demonstrates that $\mathsf { m } ^ { 6 } \mathsf { A }$ stabilizes PHR1 transcripts via a positive feedback loop. Our findings indicate that PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ modifications facilitate PSR signaling by enhancing the stability of certain mRNAs, shedding light on the role of $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications in nutrient stress responses in plants. 

The most prevalent internal covalent modification of eukaryotic messenger RNAs (mRNAs) is the methylation of adenines to produce $N ^ { 6 }$ -methyladenosine $( \mathsf { m } ^ { 6 } \mathsf { A } )$ , which occurs predominantly near the ${ } ^ { 3 ^ { \prime } }$ ends1–3 . The $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ modification is reversibly added and removed by evolutionarily conserved methyltransferases (writers) and demethylases (erasers), respectively4–8 . In Arabidopsis (Arabidopsis thaliana), 

the methylation reaction is primarily catalyzed by a multi-protein complex comprising MRNA ADENOSINE METHYLASE (MTA; ortholog of human METTL3), METHYLTRANSFERASE B (MTB; ortholog of human METTL14), FKBP12 INTERACTING PROTEIN37 (FIP37; ortholog of human WTAP), VIRILIZER (VIR), and HAKAI9–13. In a small subset of mRNAs, $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ modifications are also produced by another 

methyltransferase, FIONA1 (FIO1; ortholog of human METTL16)14–16. The $\mathsf { m } ^ { 6 } \mathsf { A }$ methyl groups can be removed by the demethylases ALKBH9B and ALKBH10B (orthologs of human ALKBH5)17–19. 

The underlying mechanisms and biological functions of $\mathsf { m } ^ { 6 } \mathsf { A }$ have been studied by profiling this modification across the transcriptomes of plants at various developmental stages and under different growth conditions2,3 . Transcriptome-wide $\mathsf { m } ^ { 6 } \mathsf { A }$ mapping usually involves immunoprecipitation of methyl-RNA combined with high-throughput RNA sequencing (MeRIP-seq or m6 A-seq); however, the fidelity of this method is compromised by the non-specific binding of the anti- $\mathsf { m } ^ { 6 } \mathsf { A }$ antibody20–22. To improve the reliability of $\mathsf { m } ^ { 6 } \mathsf { A }$ -seq, Luo and colleagues developed a method for the systematic calibration of epitranscriptomic maps using synthetic modification-free RNA library controls that resembled the endogenous transcriptome in terms of sequence and expression diversity22. This synthetic RNA library approach serves as a more robust and rigorous control22 than previously employed calibration controls, such as methyltransferase gene knockout and in vitro demodification. Although this gold-standard negative control has been used in epitranscriptome mapping studies of animals and viruses23,24, it has not previously been used for plants. 

Environmental stress dynamically alters the $\mathsf { m } ^ { 6 } \mathsf { A }$ landscape3,24,25, reflecting the important regulatory functions of this modification in the adaptation to stress. In plants, the dynamic of $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications responds to several abiotic and biotic stresses, such as salt26,27, drought28,29, heat30, cold31, excess light32, viral pathogens33,34, and fungal diseases35. However, the underlying functions of $\mathsf { m } ^ { 6 } \mathsf { A }$ in stress adaptation are not well understood. Several studies have demonstrated that stress causes dynamic $\mathsf { m } ^ { 6 } \mathsf { A }$ deposition on specific stressresponsive transcripts, which mainly affects their stability. For example, salt stress–induced $\mathsf { m } ^ { 6 } \mathsf { A }$ not only stabilizes transcripts involved in the salt-stress response27, but also destabilizes transcripts involved in the negative regulation of this response26. These contrasting effects on RNA stability were also observed in the $\mathsf { m } ^ { 6 } \mathsf { A }$ -mediated regulation of development10,17,36–38, pointing to a complex relationship between $\mathsf { m } ^ { 6 } \mathsf { A }$ modification and RNA decay. Despite these insights, a transcriptomewide RNA half-life assay with $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ profiling under normal and stress conditions has not yet been performed in plants. 

Nutrient deficiency is a major limiting factor in global crop production39,40; however, the regulation of $\mathsf { m } ^ { 6 } \mathsf { A }$ modification in the plant response to nutrition-deficiency stresses remains unclear. Phosphorus (P) is essential for all life on Earth41–43. P is a key element for photosynthesis, an important component of macromolecules such as nucleic acids, and is involved in posttranslational modifications and the biosynthesis of essential metabolites44. Plants take up P as orthophosphate (Pi) from the soil. Studies of the plant response to Pi starvation are ecologically and agriculturally important because soils in many parts of the world have low levels of available Pi due to its immobility, low solubility, or the general lack of this mineral45,46. 

To withstand environmental Pi limitation, plants have evolved a complex Pi-starvation response (PSR) comprising a series of developmental and physiological changes47,48. The evolutionarily conserved PHOSPHATE STARVATION RESPONSE1 (PHR1) and the related PHR1-like (PHL) family, collectively termed PHRs, act as central transcriptional regulators of systemic PSR signaling49–52. PHRs belong to the GARP transcription factor (TF) family and possess a MYB-like DNA-binding domain and a coiled-coil domain49,50. Upon Pi-deficiency, PHR1 forms homodimers or heterodimers with PHL1 and directly binds to a pseudopalindromic DNA sequence (GNATANC; P1BS) in the promoters of many Pi starvation–induced (PSI) genes to upregulate their expression49,53. Transcriptome analysis revealed that PHRs also indirectly regulate the expression of some Pi starvation–decreased (PSD) genes53, although the underlying mechanism remains elusive. Besides transcriptional reprogramming, PHRs regulate changes in chromatin accessibility54, lipid remodeling55, metabolic changes56, and mycorrhizal symbiosis57 in response to Pi starvation. However, it is not known whether PHRs also 

regulate $\mathsf { m } ^ { 6 } \mathsf { A }$ modification and thereby RNA stability under Pi starvation. As PHRs are fundamental regulators of the PSR, their activity is extensively regulated. PHR1 activity is inhibited through physical interactions with SPX (named after yeast Syg1 and Pho81 and mammalian XPR1) proteins in response to Pi status42,58–61. Other types of posttranslational regulation of PHR activity were also reported, including protein degradation62, sumoylation63, and phosphorylation64. However, the regulation in PHR1 transcript levels remains uncharacterized. 

Here, we performed m6 A-seq with systematic calibration using synthetic modification-free RNA libraries and produced highconfidence $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ profiles in Arabidopsis under normal and Pi-deficient stress conditions. We determined that Pi starvation induces global $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications, largely mediated by PHR proteins. Many $\mathsf { P S I - m ^ { 6 } A }$ modifications are deposited in PSR-related transcripts, including PHR1 mRNA. Although PHRs do not regulate PSI-m6 A modifications through direct transcriptional regulation, PHR1 interacts with the $\mathsf { m } ^ { 6 } \mathsf { A }$ writers MTA and MTB in nuclei under $- \mathbf { P }$ conditions. A genome-wide RNA decay assay demonstrated that PHR-mediated PSI- $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications stabilize PSR-related mRNAs. PAS-seq 2 data indicated that $\mathsf { P S I - m } ^ { 6 } \mathbf { A }$ did not impact RNA stability via alternative polyadenylation site (APA) shifts. Furthermore, mutations in putative $\mathsf { m } ^ { 6 } \mathsf { A }$ -loci in the 3′ untranslated region (UTR) of PHR1 mRNA led to shortened RNA half-lives and partial phr1 complementation, confirming that $\mathsf { m } ^ { 6 } \mathsf { A }$ stabilizes PHR1 transcripts as a positive feedback mechanism to promote PSI signaling and stress adaptation upon Pi starvation. 

# Results

# Transcriptome-wide profiling of high-confidence $\mathbf { m } ^ { 6 } \mathbf { A }$ modifications using systematic calibration with a synthetic modification-free RNA library

To explore the regulatory function of $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ in Pi homeostasis and signaling, we performed $\mathsf { m } ^ { 6 } \mathsf { A }$ -seq to examine the effect of Pi starvation on transcriptome-wide $\mathsf { m } ^ { 6 } \mathsf { A }$ levels. To correct for non-specific binding to the anti- $\mathbf { m } ^ { 6 } \mathbf { A }$ antibody during m6 A-seq, we used a synthetic modification-free RNA library to calibrate the epitranscriptomic maps22 (Fig. 1a). 

Following the protocol shown in Supplementary Fig. $\mathbf { 1 } \mathbf { a } ^ { 2 2 }$ , we replicated the entire transcriptome (all mRNAs) using T7 RNA polymerase–mediated in vitro transcription (IVT) and unmodified nucleosides (referred to as IVT RNA) (Fig. 1b). We confirmed the absence of $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications in the synthesized no- $\mathbf { \cdot m ^ { 6 } A }$ IVT RNA using liquid chromatography–mass spectrometry (LC–MS/MS) (Fig. 1c). We next performed mRNA-seq, which showed that approximately $9 5 \%$ of the genes represented in the endogenous total mRNA were also represented in the no- $\mathsf { m } ^ { 6 } \mathsf { A }$ IVT RNA. Furthermore, the global gene expression levels in the no- $\mathbf { \cdot m } ^ { 6 } \mathbf { A }$ IVT RNA library were strongly correlated with those in the endogenous transcriptome (Pearson’s $r { = } 0 . 9 5 4 )$ (Fig. 1d). The read coverages across the transcripts were also similar between the IVT RNA and endogenous mRNA libraries (Fig. 1e). These findings indicate that we successfully generated a non- $\mathbf { \cdot m } ^ { 6 } \mathbf { A }$ modified IVT RNA library that closely resembled the endogenous transcriptome in terms of sequence representation and expression. 

We generated two biological replicates of the $\mathsf { m } ^ { 6 } \mathsf { A }$ -seq and the corresponding mRNA-seq using endogenous mRNA and synthetic no-$\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ IVT RNA from seedlings grown under normal $( + \mathsf { P } )$ and Pistarvation (−P) conditions. We mapped the sequencing reads (approximately 20–28 million reads per sample) to the Arabidopsis TAIR10 reference genome and used the exomePeak pipeline to call the $\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ peaks in the transcripts of all samples using the immunoprecipitation (IP) enrichment criteria (IP/input $\geq 2$ , $\mathsf { F D R } < 0 . 0 5 )$ ). More than 14,000 peaks were identified in the endogenous mRNAs (Supplementary Fig. 1b). Most peaks $( \sim 9 0 \% )$ were represented in both biological replicates, indicating high reproducibility (Supplementary Fig. 1b). In total, 12,837 peaks and 13,379 common peaks were identified in the endogenous mRNA samples under $+ \mathsf { P }$ and $- \mathbf { P }$ conditions, 

respectively (Supplementary Fig. 1b and Supplementary Data 1). Strikingly, numerous $\mathsf { m } ^ { 6 } \mathsf { A }$ -irrelevant peaks were identified in the corresponding synthetic no- $\mathbf { \cdot m } ^ { 6 } \mathbf { A }$ IVT RNA samples, indicating the nonspecific binding of the anti $\mathbf { \cdot m } ^ { 6 } \mathbf { A }$ antibodies (Supplementary Fig. 1c, Supplementary Data 1). The peaks in the endogenous mRNA libraries were mainly located around stop codons and in $3 ^ { \prime }$ UTRs (Fig. 1f), which is consistent with a previously published $\mathsf { m } ^ { 6 } \mathsf { A }$ peak distribution1–3 . By contrast, the $\mathsf { m } ^ { 6 } \mathsf { A }$ -irrelevant peaks in the no- $\mathbf { m } ^ { 6 } \mathbf { A }$ IVT RNA libraries were mainly distributed in coding sequence (CDS) regions (Fig. 1f). Moreover, VAGGUA $\mathbf { \left( V = A / C / G \right) }$ and RGMRGR $( \mathsf { R } = \mathsf { A } / \mathsf { G }$ ; $\mathsf { M } = \mathbf { A } / \mathbf { C } )$ motifs were identified in the $\mathsf { m } ^ { 6 } \mathsf { A }$ -irrelevant peaks (Fig. 1g), which are similar to the motifs identified in the IVT RNA $\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ -seq library in a mammalian study22. 

Next, we systematically calibrated the transcriptome-wide $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications with the identified $\mathsf { m } ^ { 6 } \mathbf { A }$ -irrelevant peaks to remove falsepositive $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks from the endogenous mRNA $\mathsf { m } ^ { 6 } \mathsf { A }$ -seq libraries (Fig. 1h, see Methods). Approximately 2000 and 1500 false-positive peaks were removed from the $+ \mathsf { P }$ and $- \mathbf { P }$ samples, respectively, accounting for 11–16% of the total peaks (Fig. 1h). Consequently, we identified 10,757 and 11,861 high-confidence $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks associated with 10,420 and 11,496 genes in Col-0 (WT) seedlings under $+ \mathsf { P }$ and −P conditions, respectively (Fig. 1h and Supplementary Data 1). We also confirmed the false-positive and high-confidence $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ peaks using the Integrative Genomic Viewer (IGV), finding that false-positive $\mathsf { m } ^ { 6 } \mathsf { A }$ loci exhibited similar peaks in the WT endogenous mRNA and IVT RNA IPs, whereas the high-confidence peaks were only present in the WT mRNA IPs (Fig. 1i and Supplementary Fig. 1d). Furthermore, the highconfidence $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks exhibited stronger enrichment around stop codon and in $3 ^ { \prime }$ UTR than did the uncalibrated peaks (Fig. 1j). In addition to the plant-specific UGUA motif and common RRACH $( \mathsf { H } = \mathsf { A } /$ C/U) motif, we also identified a CAGR motif in the calibrated $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks under both $+ \mathsf { P }$ and $- \mathbf { P }$ conditions (Supplementary Fig. 1e). 

Therefore, the use of systematic calibration with a synthetic modification-free RNA library allowed us to identify high-confidence $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks under normal and Pi-deficient conditions in WT seedlings. 

# Pi starvation induces global changes in $\mathbf { m } ^ { 6 } \mathbf { A }$ levels

Under $- \mathsf { P } _ { \iota }$ , the $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ levels across the transcriptome were dramatically increased in comparison with $+ \mathsf { P }$ conditions (Fig. 2a, b). In addition, 1651 transcripts underwent de novo $\mathsf { m } ^ { 6 } \mathsf { A }$ modification in response to −P conditions, while only $5 7 5 \mathsf { m } ^ { 6 } \mathsf { A }$ -modified transcripts occurred under $+ \mathsf { P }$ conditions (Fig. 2c). We accurately identified the $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks that differed between $+ \mathsf { P }$ and $- \mathbf { P }$ conditions using the exomePeak algorithm, with the criteria of fold change $\geq 1 . 5$ and $\mathsf { F D R } < 0 . 0 5$ , excluding $\mathsf { m } ^ { 6 } \mathsf { A }$ -irrelevant peaks. Pi starvation resulted in 4418 hypermethylated $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks $\mathrm { ( P S I - m ^ { 6 } A ) }$ ) across 4364 genes compared to only 323 hypomethylated $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks $\mathbf { ( P S D \cdot m ^ { 6 } A ) }$ across 316 genes (Fig. 2d, Supplementary Data 1), indicating that Pi starvation enhances the production of $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications across the transcriptome. Compared with other published sets of genes whose $\mathsf { m } ^ { 6 } \mathsf { A }$ profiles changed in response to treatment , 19 major $\mathsf { m } ^ { 6 } \mathsf { A }$ -hypermethylation $( 7 1 . 1 \% )$ and $\mathsf { m } ^ { 6 } \mathsf { A }$ -hypomethylation $( 9 8 . 4 \% )$ in response to Pi starvation are specific (Supplementary Fig. 2a). 

These PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ and PSD- $\mathbf { m } ^ { 6 } \mathbf { A }$ peaks were mainly located near stop codons and in 3′ UTRs, although a few peaks were located at CDSs and 5′UTRs (Supplementary Fig. 2b, c). The plant-specific UGUA motif was enriched within both PSI-m6 A and $\mathsf { P S D - m } ^ { 6 } \mathbf { A }$ peaks at the 3′UTR (Supplementary Fig. 2d). Notably, an unreported CAGR motif was specifically enriched within PSI- $\mathbf { \cdot m } ^ { 6 } \mathbf { A }$ peaks at the 3′UTR, indicating a potential link of this motif with $\mathsf { P S I - m } ^ { 6 } \mathbf { A }$ function (Supplementary Fig. 2d). We performed a Gene Ontology (GO) analysis to investigate the $\mathsf { P S I - m ^ { 6 } A }$ hypermethylated genes. In addition to functions in phosphorus metabolism, the $\mathsf { P S I - m } ^ { 6 } \mathbf { A }$ genes are involved in RNA stability, RNA splicing and $3 ^ { \prime }$ -end processing, rRNA processing, tRNA modification, and histone and DNA modification pathways, which are indicative of 

$\mathsf { m } ^ { 6 } \mathsf { A }$ -mediated regulations of RNA metabolism and transcription in response to Pi starvation (Fig. 2e). PSI- $\mathsf { m } ^ { 6 } \mathsf { A }$ genes are also involved in photosynthesis, defense responses, protein transmembrane transport, and protein lipidation (Fig. 2e). By contrast, PSD-m6 A hypomethylated genes are involved in responses to stress, inorganic anion transport, and sulfur compound biosynthesis (Fig. 2f). These results suggest that $\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ RNA modifications are involved in regulating multiple processes during the PSR, from transcription to RNA metabolism and ion homeostasis. 

We then focused on the PSR-related genes. Given that Pi starvation triggers the altered expression of thousands of genes53, we assessed whether the changes in $\mathsf { m } ^ { 6 } \mathsf { A }$ modification were associated with changes in the expression of PSR genes. Consistent with previous studies53, our RNA-seq data showed that thousands of genes were upregulated $( - \mathsf { P } / + \mathsf { P } \ge 1 . 5$ , $P { < } 0 . 0 5$ ; PSI genes) or downregulated $( + \mathsf { P } / ^ { }$ $- \mathsf { P } \ge 1 . 5$ , $P { < } 0 . 0 5$ ; PSD genes) by Pi-deficiency (Supplementary Fig. 2e, Supplementary Data 2). Notably, approximately one-third of the PSI genes and more than $70 \%$ of the PSD genes contained $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ modifications (Supplementary Fig. 2f). Furthermore, a joint analysis of the $\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ -seq and RNA-seq data revealed a large number of $\mathsf { m } ^ { 6 } \mathsf { A }$ -hypermethylated PSD genes (638 PSI-m6 A & PSD genes, accounting for ${ \sim } 7 1 \%$ of the total $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ -altered PSR genes). These results suggest that $\mathsf { m } ^ { 6 } \mathsf { A }$ modification might regulate the RNA levels of PSD genes in response to Pi starvation (Fig. 2g and Supplementary Data 1). 

Furthermore, the transcripts of several known Pi homeostasis and PSR-signaling genes harbored $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ modifications. These genes included the central PSR-signaling gene PHR1 and its homologs49,53, the PSRsignaling repressor genes SPX2 and $S P X 4 ^ { 5 8 , 6 5 }$ , the phosphate transporter trafficking facilitator gene PHOSPHATE TRANSPORTER TRAFFIC FACILITATOR1 $( P H F I ) ^ { 6 6 , 6 7 }$ , the Pi efflux transporter gene $P H O I ^ { 6 8 }$ , the root tip low-phosphate sensor LOW PHOSPHATE ROOT2 $( L P R 2 ) ^ { 6 9 }$ , the diphosphoinositol pentakisphosphate kinase genes VIP1 HOMOLOG1 (VIH1) and $V I H 2 ^ { 7 0 }$ , and the vacuolar Pi transporter genes PHOSPHATE TRANSPORTER 5;1 (PHT5;1) and $P H T 5 ; 3 ^ { 7 1 , 7 2 }$ (Fig. 2h, i and Supplementary Fig. 2g). The GO results also showed PSR-related pathways represented by $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ -modified PSI genes, regardless of PSI- $\mathbf { \cdot m } ^ { 6 } \mathbf { A }$ (Supplementary Fig. 2h, i). 

Together, the above data show that Pi starvation induces global changes in $\mathsf { m } ^ { 6 } \mathsf { A }$ levels. Many PSR-related transcripts were $\mathsf { m } ^ { 6 } \mathsf { A }$ modified, suggesting that $\mathsf { m } ^ { 6 } \mathsf { A }$ regulates Pi homeostasis and PSR signaling.

# PHRs do not mediate PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ modification through direct transcriptional regulation

The PHR TFs regulate the expression of many PSI genes53; thus, we investigated whether the PHRs mediate the $\mathsf { m } ^ { 6 } \mathsf { A }$ levels of these PSI- $\mathsf { m } ^ { 6 } \mathsf { A }$ transcripts via the direct transcriptional regulation of these genes. We first compared the PHR-mediated $\mathsf { P S I - m ^ { 6 } A }$ genes with direct target genes of PHR1 that were previously identified using chromatin immunoprecipitation (ChIP)-seq73. Only 155 of the PHR-mediated PSI-$\mathsf { m } ^ { 6 } \mathsf { A }$ genes (accounting for $8 \%$ of the total) overlapped with the direct target genes of PHR1 (accounting for $7 \%$ of the total) (Supplementary Fig. 4a), indicating that the PHR1-mediated regulation of $\mathsf { P S I - m ^ { 6 } A }$ modifications is mainly achieved via a mechanism other than direct transcriptional regulation. To further explore this idea, we compared our RNA-seq data from WT and phr1 phl1. More than half of the PSI genes $( \sim 5 1 \% )$ were repressed (classified as PHR-controlled PSI genes), while only ${ \sim } 1 5 \%$ of PSD genes were upregulated (classified as PHRcontrolled PSD genes) in phr1 phl1 compared with the WT under −P conditions (Supplementary Fig. 4b, c and Supplementary Data 2). This is consistent with the previous finding that PHRs mainly transcriptionally regulate the induction of PSI genes under −P conditions53. However, only 34 of the PHR-mediated $\mathsf { P S I - m } ^ { 6 } \mathbf { A }$ genes (accounting for $2 \%$ of the total) overlapped with the PHR-controlled PSI genes (accounting for $3 \%$ of the total) (Supplementary Fig. 4d), confirming the notion that PHRs do not transcriptionally regulate PHR-mediated $\mathsf { P S I - m } ^ { 6 } \mathbf { A }$ genes. 

In addition, we measured the expression levels of genes encoding $\mathsf { m } ^ { 6 } \mathsf { A }$ writers (MTA, MTB, FIP37, VIR, HAKAI, and FIONA1) and erasers (ALKBH9B and ALKBH10B) in the WT and phr1 phl1 under $+ \mathsf { P }$ and −P conditions. The expression of these genes was not greatly altered in response to Pi-deficiency or the mutation of PHR1 and PHL1 (Supplementary Fig. 4e). We further detected the protein levels of MTA in the WT and phr1 phl1 mutants under $+ \mathsf { P }$ and $- \mathbf { P }$ but found no obvious changes in total proteins (Supplementary Fig. 4f). Besides, the amount of MTA and MTB proteins in nuclei was comparable with the WT between $+ \mathsf { P }$ and $- \mathbf { P }$ conditions (Supplementary Fig. 4g). 

Taken together, these data demonstrate that PHRs mediate PSI-$\mathsf { m } ^ { 6 } \mathsf { A }$ modifications, but not via either direct transcriptional regulation or protein abundance changes of $\mathsf { m } ^ { 6 } \mathsf { A }$ writers, pointing to the involvement of an unknown regulatory mechanism. 

# PHR1 interacts with MTA and MTB in nuclei under Pi starvation conditions

Several factors interact with $\mathsf { m } ^ { 6 } \mathsf { A }$ writers to regulate miRNA biogenesis74,75, the circadian clock76 and chlorophyll homeostasis77. Using yeast two-hybrid (Y2H) assays, we observed the interactions of PHR1 with MTA and MTB, but not with FIP37, another component of the core $\mathsf { m } ^ { 6 } \mathsf { A }$ methyltransferase complex (MTC) (Fig. 4a). Meanwhile, SPX1 and SPX2, which are repressors of PHR activity, did not interact with any MTC component in the Y2H system (Fig. 4b). To confirm the in vivo interaction between PHR1 and MTA/MTB, we performed bimolecular fluorescence complementation (BiFC) assays (Fig. 4c). 

PHR1 interacted with MTA and MTB in the nucleus, and they formed speckles in some cells (Fig. 4c). These results are consistent with reports of condensate formation by the $\mathsf { M T C } ^ { 7 4 , 7 5 }$ . Furthermore, we validated the PHR1-MTA/MTB interaction in plants under −P conditions by nuclear Co-IP (Fig. 2d). 

These results elucidated the association of PHR1 with MTA and MTB under $- \mathbf { P }$ conditions in plants. Given the current data obtained under $- \mathbf { P }$ conditions in plants, it appears that PHR1, a key transcription factor in the PSR, plays a previously unrecognized role in regulating PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ under −P, possibly through interacting with MTA and MTB. 

# $\mathbf { m } ^ { 6 } \mathbf { A }$ RNA modification facilitates Pi homeostasis and systemic PSR signaling

To further decipher the roles of $\mathsf { m } ^ { 6 } \mathsf { A }$ modification in Pi homeostasis and PSR signaling, we developed a line hypomorphic for the $\mathsf { m } ^ { 6 } \mathsf { A }$ writer gene MTA using the artificial microRNA method (amiR-mta) (Fig. 5a and Supplementary Fig. 5a). The expression level of MTA in amiR-mta was repressed to ${ \sim } 2 5 \%$ that of WT plants (Fig. 5f and Supplementary Fig. 5b). We quantified the global $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ level in amiR-mta using LC–MS/ MS, revealing a dramatic reduction in the $\boldsymbol { \mathrm { m } } ^ { 6 } \mathbf { A } / \mathbf { A }$ ratio compared with the WT (decreased to ${ \sim } 4 0 \%$ ) under both $+ \mathsf { P }$ and $- \mathbf { P }$ conditions (Supplementary Fig. 5c). 

The amiR-mta plants exhibited inhibited growth under $+ \mathsf { P }$ conditions, with reduced biomass and shorter primary roots, which is consistent with previous findings (Fig. ${ 5 { \bf { a } } - { \bf { c } } } ) ^ { 1 0 , 1 3 }$ . Under $- \mathbf { P }$ stress, this growth inhibition of amiR-mta was deteriorated (Fig. 5a–c). We then investigated the Pi homeostasis- and PSR-related physiological traits of the WT and amiR-mta. Compared with the WT, the knockdown of MTA resulted in a significant reduction in the shoot Pi concentration under both $+ \mathsf { P }$ and −P conditions, indicating that the $\mathsf { m } ^ { 6 } \mathsf { A }$ modification plays a positive role in Pi homeostasis (Fig. 5d). In addition, the inhibition of primary root growth is an important trait of the local $\mathsf { P S R } ^ { 4 7 , 6 9 }$ . The ratio of primary root length under $- \mathbf { P }$ vs. $+ \mathsf { P }$ conditions was slightly smaller in amiR-mta than in the WT, suggesting that the $\mathsf { m } ^ { 6 } \mathsf { A }$ modification moderately represses the local PSR (Fig. 5e). These physiological phenotypes caused by the knockdown of the $\mathsf { m } ^ { 6 } \mathsf { A }$ writer gene MTA provide further evidence that $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications play a regulation role in Pi homeostasis and the PSR. 

We further explored the involvement of the $\mathsf { m } ^ { 6 } \mathsf { A }$ modification in PHR-mediated systemic PSR signaling. We used RT-qPCR to assess the steady-state transcript levels of several systemic PSR-signaling genes, including PHR1 and the downstream Pi transport- and Pi metabolismrelated PSI genes. The steady-state transcript levels of PHR1, PHF1, PHO1, PHT1;1/PT1, and ACID PHOSPHATASE 5 (ACP5) showed moderate but significant reductions in amiR-mta compared with the WT under −P (Fig. 5f). This is consistent with the reduced Pi concentration of amiR-mta plants (Fig. 5d). To further investigate the enhanced functions of $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications in PSR and Pi homeostasis in plants under $- \mathbf { P }$ conditions, we treated 7-day-old WT plants grown under normal conditions with the chemical STM2457, an inhibitor of $\mathsf { m } ^ { 6 } \mathsf { A }$ writers78 for 7 days under −P stress. Compared with 14-day-old WT plants grown under normal conditions, both STM2457-treated and mock WT plants exhibited typical morphological responses under Pi deficiency, such as inhibited primary root growth and anthocyanin accumulation (Supplementary Fig. 5d). Notably, the Pi concentration in shoots was lower in STM2457-treated WT plants than in mock plants under $- \mathbf { P }$ conditions (Supplementary Fig. 5e). Moreover, the expression levels of PHR1 and its downstream PSR signaling genes ACP5 and PHF1 were lower in STM2457-treated WT plants compared with mock plants (Supplementary Fig. 5f). These results support the hypothesis that $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications positively regulate Pi homeostasis and PHR-mediated systemic PSR signaling in plants under Pi-deficiency stress.

# Feedback regulation of PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ promotes the stability of PSD gene transcripts

We next evaluated whether $\mathsf { P S I - m ^ { 6 } A }$ modifications participate in regulating the stability of PSR transcripts. We compared RNA half-life data from WT and amiR-mta plants. Under $+ \mathsf { P }$ conditions, the RNA half-life of the majority of genes was higher in amiR-mta compared with the WT, with 11,193 genes showing a longer half-life and 2965 genes showing a shorter half-life in the amiR-mta plants (Supplementary Fig. 7 and Supplementary Data 3). These results suggest that $\mathsf { m } ^ { 6 } \mathsf { A }$ plays dual roles in RNA stability and that $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications mainly destabilize transcripts under normal conditions. Moreover, the half-lives of both $\mathsf { m } ^ { 6 } \mathsf { A }$ -methylated and non- $\mathsf { m } ^ { 6 } \mathsf { A }$ -methylated transcripts increased globally in amiR-mta under $+ \mathsf { P }$ conditions, indicating that $\mathsf { m } ^ { 6 } \mathsf { A }$ negatively regulates global RNA stability in both direct and indirect manners (Supplementary Fig. 7d). 

Unexpectedly, under $- \mathbf { P }$ conditions, the RNA half-life did not exhibit a dramatic global increase in amiR-mta compared to the WT (Supplementary Fig. 7a). A detailed analysis showed that 6804 and 5336 genes had increased and decreased half-lives, respectively (Fig. 6e and Supplementary Data 3). The positive effect of $\mathsf { m } ^ { 6 } \mathsf { A }$ on RNA stability appears to be enhanced under $- \mathbf { P }$ conditions. We examined the RNA stability of the −P-mediated $\mathsf { m } ^ { 6 } \mathsf { A } ^ { . }$ -hypermethylated PSR genes. The half-lives of $\mathsf { m } ^ { 6 } \mathsf { A }$ -hypermethylated PSD genes (PSI-m6 A & PSD genes), but not $\mathsf { m } ^ { 6 } \mathsf { A }$ -hypermethylated PSI genes (PSI-m6 A & PSI genes), significantly increased under −P compared to control conditions $( + \mathsf { P } )$ , indicating that PSI-m6 A-mediated feedback enhances the RNA stability of $\mathsf { m } ^ { 6 } \mathsf { A }$ -hypermethylated PSD transcripts (Fig. 6f). Furthermore, this increased RNA stability of −P-induced $\mathsf { m } ^ { 6 } \mathsf { A }$ -hypermethylated PSD genes was repressed in amiR-mta plants (with dramatically reduced $\mathsf { m } ^ { 6 } \mathsf { A }$ levels of global and $\mathsf { P S I - m } ^ { 6 } \mathbf { A }$ & PSD genes [Supplementary Fig. 8a, b]) compared with the WT under $- \mathbf { P }$ but not $+ \mathsf { P }$ conditions, confirming the positive role of $\mathsf { P S I - m ^ { 6 } A }$ in the RNA stability of PSD genes (Fig. 6g). Moreover, more than half of these $\mathsf { m } ^ { 6 } \mathsf { A }$ -hypermethylated PSD genes were also PHR-mediated $\mathsf { P S I - m } ^ { 6 } \mathbf { A }$ genes (Fig. 6h). We named these overlapping genes PHR & $\mathsf { m } ^ { 6 } \mathsf { A } \cdot$ -controlled PSD genes (Supplementary Data 1). The half-lives of the PHR & $\mathsf { m } ^ { 6 } \mathsf { A }$ -controlled PSD genes also increased in response to Pi starvation in the WT but not in amiR-mta plants (Fig. 6i), indicating that feedback regulation of the PHRs fine-tunes the expression of some PSD genes via $\mathsf { m } ^ { 6 } \mathsf { A } \cdot$ -mediated mRNA stabilization. 

Taken together, our transcriptome-wide RNA stability profiling of WT and amiR-mta plants under $+ \mathsf { P }$ and $- \mathbf { P }$ conditions, as well as the $\mathsf { m } ^ { 6 } \mathsf { A }$ landscape in miR-mta under $- \mathbf { P }$ conditions, revealed a new layer of PSR regulation at the level of RNA stability. Moreover, this analysis revealed that PHR-mediated Pi starvation–induced $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ feedback extends the RNA half-lives of PSD genes. 

# PSI-m6 A does not facilitate RNA stability through APA shifts

It has been reported that $\mathsf { m } ^ { 6 } \mathsf { A }$ could affect RNA stability via APA shifts26,82. To investigate whether PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ facilitates RNA stability via APA shifts, we performed PAS-seq $2 ^ { 8 3 }$ to profile APA sites on a transcriptome-wide level under $+ \mathsf { P }$ and $- \mathbf { P }$ conditions in WT and amiR-

mta seedlings. PCA revealed high reproducibility among three biological replicates of each sample (Supplementary Fig. 9a). Moreover, most poly(A) tags (PATs) aligned to the 3′UTRs of protein-coding genes $( \sim 7 0 \% )$ (Supplementary Fig. 9b). These results reflect the high quality of our PAS-seq 2 data. 

Following previous studies82,84, we assigned adjacent PATs within a 24-nt window to PAS clusters (PACs). In total, 71,753 PACs (corresponding to 17,968 genes) were detected in WT and amiR-mta under −P and $+ \mathsf { P }$ conditions. Furthermore, we compared the PAC shift events between the WT and amiR-mta. Under both $+ \mathsf { P }$ and −P conditions, the knockdown of MTA led to dramatic PAC shifts (Supplementary Fig. 9c). Specifically, 2463 significant distal-to-proximal shifts (hereafter d2p) and 461 significant proximal-to-distal shifts (hereafter p2d) were detected under $+ \mathsf { P }$ conditions and 1992 significant d2p and 397 significant p2d events were detected under –P conditions in amiR-mta compared with the WT (Supplementary Fig. 9c and Supplementary Data 4). This observation is also similar to the results for the $\mathsf { m } ^ { 6 } \mathsf { A }$ reader mutant cpsf30-l (cleavage and polyadenylation specificity factor 30 long-isoform) 82. Approximately $54 \%$ of d2p-shifted genes (total of 1222) and $14 \%$ of p2d-shifted genes (total of 657) in cpsf30-l are also shifted in amiR-mta (Supplementary Fig. 9d). These results support the role of $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications in regulating APA. 

Next, we investigated the −P-induced PAC shifts in WT. We identified 203 significant d2p and 189 significant p2d events under −P compared with $+ \mathsf { P }$ in the WT, pointing to the potential regulation of APA shifts in response to Pi deficiency (Supplementary Fig. 9c and Supplementary Data 4). Approximately half of these PSR APA-shifted genes were accompanied by changes in RNA stability (Supplementary Fig. 9e). We compared these PSR $t _ { 1 / 2 }$ -altered and APA-shifted genes with PSR $\mathsf { m } ^ { 6 } \mathsf { A }$ -altered genes. We observed a non-significant overlap of PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ and PSD- $\mathsf { m } ^ { 6 } \mathsf { A }$ genes with PSR $t _ { 1 / 2 }$ -altered d2p-shifted and p2dshifted genes (Supplementary Fig. 9f). These overlapping genes only accounted for ${ \bf - 1 \% }$ of PSI- $\mathbf { \cdot m } ^ { 6 } \mathbf { A }$ genes, indicating that the effects of PSI-$\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ modifications on RNA stability are not due mainly to APA shifts (Supplementary Fig. 9f). Meanwhile, the APA of only $1 8 . 5 \%$ of PSI- $\mathbf { \cdot m ^ { 6 } A } \&$ PSD genes was altered in amiR-mta compared with WT under −P conditions (Supplementary Fig. 9g). Moreover, APA shift and IGV analysis showed that the APA of PHR1 transcripts was not obviously shifted in the WT or amiR-mta under $+ \mathsf { P }$ or $- \mathbf { P }$ conditions (Supplementary Fig. 9h). 

Our results showed a potential regulation in APA shifts in response to Pi deficiency, but the PSI- $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ -induced increase in RNA stability is not due to shifts in APA.

# Discussion

There is mounting evidence that widespread $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ modifications play critical roles in the post-transcriptional regulation of RNA metabolism and physiological processes1–3 . Plants are frequently exposed to nutrient-deficient stress, leading to widespread changes in gene expression39,40. However, the regulatory function of $\mathsf { m } ^ { 6 } \mathsf { A }$ in responses to these physiological stresses remains elusive. In this study, we examined the $\mathsf { m } ^ { 6 } \mathbf { A } \cdot$ -methylome, the transcriptome, genome-wide RNA decay, global profiling of APA sites, PHR1-MTA/MTB protein interactions, and the effects of mutating $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ loci to dissect the role of $\mathsf { m } ^ { 6 } \mathsf { A }$ -mediated regulation of RNA stability in facilitating systemic PSR signaling and Pi homeostasis under Pi-deficient stress. We demonstrated that PHR-mediated PSI- $\mathbf { m } ^ { 6 } \mathbf { A }$ modifications not only promote stability of some PSD transcripts, but also stabilize PHR1 mRNA to facilitate the expression of PSI genes via a feedback loop (Fig. 7h). 

PHR1 and its family members are central TFs in the transcriptional regulation of the $\mathsf { P S R } ^ { 4 9 , 5 3 }$ . Here, we showed that PHRs mediate epitranscriptomic regulation during Pi homeostasis and the PSR. We conducted m6 A-seq with systematic calibration using synthetic modification-free RNA libraries to remove false-positive peaks and obtained high-confidence $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ landscapes under normal and Pideficient stress conditions (Fig. 1). Pi starvation induced widespread $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications, many of which were added to Pi homeostasis- and PSR-related transcripts (Fig. 2). We also showed that PHR proteins mediate PSI- $\mathsf { m } ^ { 6 } \mathsf { A } _ { i }$ , as these global modifications were diminished in the phr1 phl1 double mutant under −P conditions (Fig. 3). However, the PHRs do not transcriptionally regulate the expression of either the PSI-$\mathsf { m } ^ { 6 } \mathsf { A }$ -marked mRNAs or the known $\mathsf { m } ^ { 6 } \mathsf { A }$ methyltransferases or demethylases (Supplementary Fig. 4), raising the interesting question of how PHRs mediate PSI- $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications. Blue light induces the liquid–liquid phase separation of CRYPTOCHROME2 (CRY2) to cocondense the $\mathsf { m } ^ { 6 } \mathsf { A }$ writer proteins MTA, MTB, and FIP37, thereby promoting the addition of $\mathsf { m } ^ { 6 } \mathsf { A }$ to transcripts of the central circadian clock oscillator genes76. CRY2 also co-condenses another $\mathsf { m } ^ { 6 } \mathsf { A }$ writer protein, FIO1, to promote the methylation of transcripts encoding multiple regulators of chlorophyll homeostasis77. In addition, the histone modification H3K36me3 can guide co-transcriptional $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications in mammals by interacting with METTL1486. Analogously, H3K36me2 is highly correlated with $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications in Arabidopsis87. Although a limitation exists in that the interaction between PHR1 and MTA/MTB in plants under $+ \mathsf { P }$ conditions remains uninvestigated, we observed that PHR1 interacts with MTA and MTB in nuclei under −P conditions (Fig. 4). It is plausible that PHRs might act as $" \mathbf { m } ^ { 6 } \mathbf { A }$ recruiters” to guide $\mathsf { m } ^ { 6 } \mathsf { A }$ writers to certain transcripts, facilitating 

PSR-related $\mathsf { m } ^ { 6 } \mathsf { A }$ methylation upon Pi starvation. These findings expand our understanding of the regulatory functions of PHRs beyond the transcriptional level to the epitranscriptional level. 

The $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications regulate almost all aspects of mRNA metabolism, including transcription, pre-mRNA processing, folding, translation, and decay1–3 . To date, the best-characterized role of $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ in various plant species is its influence on mRNA stability, which affects multiple developmental processes and stress responses2,3,88. $\mathsf { m } ^ { 6 } \mathsf { A }$ exerts positive or negative effects on specific transcripts under certain distinct conditions3,88. In some studies, the combination of $\mathsf { m } ^ { 6 } \mathsf { A }$ -seq and RNA-seq led to apparently paradoxical conclusions about the effect of $\mathsf { m } ^ { 6 } \mathsf { A }$ on RNA abundance3,27,36,89; however, these studies did not directly and globally investigate the relationship between mRNA halflife and $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications in plants. We performed transcriptomewide RNA decay assays in the WT and a line in which the $\mathsf { m } ^ { 6 } \mathsf { A } ^ { \prime }$ -writer gene MTA is suppressed (amiR-mta) under normal and Pi-deficient stress conditions (Fig. 6 and Supplementary Fig. 7). Intriguingly, our results exhibited a distinct effect of $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications on overall mRNA stability, in which $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ mainly destabilizes transcripts under normal growth conditions but stabilizes them under Pi-deficient conditions (Fig. 6 and Supplementary Fig. 7). Our results confirm the dual effects of $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ on mRNA stability, indicating that $\mathsf { m } ^ { 6 } \mathsf { A }$ -mediated RNA decay is dependent on the environment, as exemplified by $+ \mathsf { P }$ and $- \mathbf { P }$ conditions. 

We showed that $\mathrm { P S I } { \cdot } \mathbf { m } ^ { 6 } \mathbf { A }$ modifications mediate the regulation of RNA stability in response to Pi starvation. The effects of PSR were previously not well understood at the RNA level; however, our genome-wide RNA decay assay revealed the previously uncharacterized regulation of RNA stability under Pi-deficient stress, in which the half-lives of more than half of transcripts are altered by Pi starvation (Fig. 6c). Approximately 2000 PSD genes show decreased expression levels under Pi starvation (Supplementary Fig. 2e)53, but it is unclear how this is regulated. Our results show that PHR-controlled $\mathsf { P S I - m ^ { 6 } A }$ fine-tunes the expression of these PSD genes by stabilizing their transcripts (Figs. 5 and 6). Moreover, we demonstrated that $\mathsf { m } ^ { 6 } \mathsf { A }$ modifications stabilize PHR1 mRNA to facilitate PSI signaling (Fig. 7). Although the steady-state level of PHR1 transcript is comparable or moderately increased by $- \mathsf { P } ^ { 4 9 }$ , the mechanism by which its expression is maintained had been unknown. Our findings provide evidence for the post-transcriptional stabilization of PHR1 mRNA by $\mathsf { m } ^ { 6 } \mathsf { A }$ upon Pi starvation, promoting the downstream signaling pathway. 

The above results raise an interesting question about the mechanism of PSI- $\mathsf { m } ^ { 6 } \mathsf { A }$ -mediated RNA decay under −P conditions. $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ reader proteins bind to the $\mathsf { m } ^ { 6 } \mathsf { A }$ modification and determine the fate of $\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ -methylated transcripts in diverse RNA metabolic pathways. In mammals, the binding of the $\mathsf { m } ^ { 6 } \mathsf { A }$ reader YTHDF2 to $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ -modified transcripts promotes their decay90,91, while the binding of other $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ readers (IGF2BP proteins) stabilizes $\mathsf { m } ^ { 6 } \mathsf { A }$ -modified transcripts92. In Arabidopsis, the $\mathsf { m } ^ { 6 } \mathsf { A }$ reader EVOLUTIONARILY CONSERVED C-TERMINAL REGION2 (ECT2) binds to and promotes the stability of transcripts related to trichome morphogenesis93. The $\mathsf { m } ^ { 6 } \mathsf { A }$ reader CPSF30-L regulate the alternative polyadenylation of some nitrate signaling–related gene transcripts84. First, our PAS-seq 2 data showed that the impact of PSI- $\mathbf { \cdot m } ^ { 6 } \mathbf { A }$ did not occur through APA shifts (Supplementary Fig. 9). Second, we did not find that PHR1 was the target of ECT2 in published FA-CLIP-seq data93. In addition, a preprint manuscript indicates that the abundance of the $\mathsf { m } ^ { 6 } \mathsf { A }$ reader ECT4 is reduced under Pi-deficiency stress94. We further detected the phenotype of ect2/3/4 under −P stress. Consist with a recent study95, the primary root length was reduced in ect2/3/4 compared with the WT under −P treatment (Supplementary Fig. 12a, b). However, the fresh weight and Pi concentration in shoots were comparable between the WT and ect2/ 3/4 under −P conditions (Supplementary Fig. 12c, d). As expected, PHR1 expression and mRNA stability were unchanged in ect2/3/4 versus the WT under −P conditions (Supplementary Fig. 12e, f). These results 

indicated that $\mathsf { m } ^ { 6 } \mathsf { A }$ readers $\operatorname { E C T } 2 / 3 / 4$ do not participate in PSI-$\mathsf { m } ^ { 6 } \mathsf { A }$ -mediated PHR1 stabilization. 

Our results suggest that unknown PSR $\mathsf { m } ^ { 6 } \mathsf { A }$ reader(s) modulate $\mathsf { P S I - m } ^ { 6 } \mathbf { A }$ -mediated mRNA stabilization. It would therefore be interesting to systematically investigate the activities of $\mathsf { m } ^ { 6 } \mathsf { A }$ readers upon Pi starvation. 

In conclusion, we analyzed the $\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ -methylome, transcriptome, PHR1-MTA/MTB protein interactions, global APA site profiling, and mRNA decay landscape (validated by m6 A-loci mutagenesis) to systematically investigate the role of the $\mathsf { m } ^ { 6 } \mathsf { A }$ modification in the PSR. Our findings demonstrate that PHR-mediated stress-induced $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ modifications play a crucial role in maintaining Pi homeostasis and enhancing systemic PSR signaling by stabilizing PSR-related transcripts. These findings uncover how plants respond to nutritiondeficient stress at the epitranscriptome level. 

# Methods

# Plant materials and growth conditions

Seeds of Arabidopsis (Arabidopsis thaliana) accession Columbia (Col-0) and the phr1 (SALK_067629C), phl1 (CS832612), and ect2/3/4 (CS2110132) T-DNA null mutants were obtained from the Arabidopsis Biological Resource Center (https://abrc.osu.edu/). The phr1 phl1 double mutant was obtained by crossing phr1 with the phl1 single mutant. To generate amiR-mta plants, Col-0 plants were transformed with the binary vector pBA-P35S:amiR-mta by the floral dip method96. Similarly, for PPHR1:gPHR1-FLAG-4XMYC-3UTR-WT; phr1 and PPHR1:gPHR1-FLAG-4XMYC-3UTR-MUT; phr1 plants, homozygous phr1 mutants were transformed with the binary vectors pBA002a-PPHR1:gPHR1-FLAG-4XMYC-3UTR-WT or pBA002a-PPHR1:gPHR1-FLAG-4XMYC-3UTR-MUT by the floral dip method96. 

The wild type (Col-0), mutants, and transgenic lines were grown under a 12-h-light/12-h-dark photoperiod at $2 2 ^ { \circ } \mathrm { C }$ at a relative humidity of ${ \sim } 5 0 \%$ . For Pi starvation treatment, seeds were germinated and grown on Murashige and Skoog (MS) agar medium supplemented with $1 \%$ sucrose for seven days. The seven-day-old plants were transferred from MS agar medium to fresh MS agar medium containing $1 . 2 5 \mathsf { m M }$ Pi $( + \mathsf { P } )$ or $6 . 2 5 \mu \mathrm { M }$ Pi (−P) and incubated for seven more days.

# $\mathbf { m } ^ { 6 } \mathbf { A }$ -seq data analysis

Approximately 23 million paired-end 150 bp (PE150) reads were obtained for each sample. Raw reads were subjected to adapter trimming and filtering of low-quality reads (quality $_ { < 2 0 }$ ) using Cutadapt (v4.1) software99. Basic quality control was conducted using FastQC (v0.11.9) (www.bioinformatics.babraham.ac.uk/projects/fastqc/), and reads ${ < } 1 5$ bp long were discarded. Quality-filtered reads were mapped to the Arabidopsis genome (TAIR10) using HISTA2 $( \mathsf { v } 2 . 2 . 1 ) ^ { 1 0 0 }$ with the parameter ‘–rna-strandness RF’. Mapped reads were visualized in Integrative Genomics Viewer $( \mathsf { I G V } ) ^ { 1 0 1 }$ after RPKM (Reads Per Kilobase per Million mapped reads) normalization and used for the following analyses. 

First, $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks were called using the R package exomePeak (v2.16.0) (bioconductor.org/packages//2.13/bioc/html/exome-Peak.html) using the enrichment criteria (IP/Input) $\geq 2$ and $\mathsf { F D R } < 0 . 0 5$ . Systematic calibration using modification-free IVT RNA libraries was then performed as previously described22. Briefly, false-positive peaks were defined as the overlapping/common peaks sharing more than $50 \%$ fractions of peak regions in the endogenous mRNA library and modification-free IVT RNA library. BEDtools (v2.30.0) intersect102 was used to find the overlapping peaks. The calibrated high-confidence $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks were defined as peaks that were specifically identified in both biological replicates in mRNA $\mathsf { m } ^ { 6 } \mathsf { A }$ -seq libraries but not in IVT RNA $\mathsf { m } ^ { 6 } \mathsf { A }$ libraries. The differential $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks were then identified using exomePeak based on the criteria of fold-change $\geq 1 . 5$ and FDR < 0.05. The differential $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks that overlapped with known falsepositive $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks were removed. Peak annotation was performed using BEDtools $( \mathsf { v } 2 . 3 0 . 0 ) ^ { 1 0 2 }$ . HOMER (v4.11) software (homer.ucsd.edu/ homer/motif) was used to identify $\mathsf { m } ^ { 6 } \mathsf { A }$ peak motifs in the calibrated high-confident $\mathsf { m } ^ { 6 } \mathsf { A }$ peaks and peak motifs in peaks from the IVT RNA libraries.

# PAS-seq 2 and data analysis

Poly(A) sequencing (PAS-seq) 2 was performed as described previously83. In brief, total RNA was extracted from 14-day-old Arabidopsis seedlings using TRIzol™ reagent (ThermoFisher) according to the manufacturer’s instructions. Poly(A) RNA was enriched using Dynabeads Oligo $( \mathsf { d T } ) _ { 2 5 }$ beads (ThermoFisher). The enriched RNA was fragmented using RNA Fragmentation Reagents (ThermoFisher, AM8740) at $7 0 ^ { \circ } \mathrm { C }$ for 5 min. Next, fragmented mRNAs were subjected to first-strand cDNA synthesis by SuperScript III reverse transcriptase (ThermoFisher) with the PAS-seq 2 oligo(dT) primer (GTGACT GGAGTTCCTTGGCACCCGAGAATTCCATTTTTTTTTTTTTTTTTTTTV, where V represents A/C/G) and a template-switch oligo (TSO) containing locked nucleic acid (LNA) (CTACACGACGCTC TTCCGATCTCATrGrG+G). The cDNA was purified with AMPure XP beads (Beckman) and amplified with Illumina TruSeq Universal adapter and RNA PCR indexed primer. Amplified products were gel extracted to isolate fragments of 185–225 bp. Purified fragments were sequenced on an Illumina sequencing platform. 

Data analysis was performed as described in previous studies84. In brief, raw reads were trimmed to remove adapters and filtered for lowquality bases $( Q < 2 0 )$ using Cutadapt (v4.1). The filtered reads were aligned to the Arabidopsis thaliana genome (TAIR10) using STAR (v2.7.10). Internal priming artifacts, defined as six or more continuous As or seven or more As in a 10-nt window downstream from the poly(A) junction, were discarded. Next, polyadenylation sites (PASs) located within a 24-nt window and with $\mathbf { T } \mathbf { P } \mathbf { M } \geq 3$ in at least two biological replicates were clustered into a PAS cluster (PAC). The PAS shift events between indicated samples were compared with Fisher’s exact test. A pair of PAS shifts with $\mathsf { F D R } < 0 . 0 1$ was considered significant. For genes with more than two significantly shifted PACs, the two most dominant differential PACs were selected to represent the shift orientation of the given genes.

# Transcriptome-wide RNA decay analysis

The analysis of transcriptome-wide RNA decay was performed as described79 (bioconductor.org/packages/release/bioc/html/RNAdecay.html) with some modifications. Pi starvation-treated (−P) or untreated control $( + \mathsf { P } )$ two-week-old Col-0 (WT) and amiR-mta seedlings were harvested and placed in incubation buffer containing 15 mM sucrose, 1 mM KCl, 1 mM PIPES pH 6.25, and 1 mM sodium citrate. The transcriptional inhibitor cordycepin was then added to the incubation buffer to a final concentration of 1 mM. Vacuum infiltration (~ 12 psi) 

was conducted for 1 min to facilitate sample penetration. Initial (0 min, $\mathsf { T } _ { 0 } )$ samples were immediately harvested and flash-frozen in liquid nitrogen. Then, vacuum infiltration (~ 12 psi) was performed twice for 1 min each time. Subsequently, samples were collected at 15, 30, 60, and 120 min following the first vacuum release, flash-frozen in liquid nitrogen, and ground to a powder. Three biological replicates were conducted for each time point and each material. Total RNA was extracted from the samples using TRIzol (ThermoFisher) and used to prepare libraries with a Stranded mRNA-seq Lib Prep Module for Illumina (ABclonal). The libraries were sequenced as 150-bp paired-end reads on an NovaSeq 6000 instrument at Annoroad. 

The raw sequencing reads from the RNA decay RNA-seq were subjected to adapter trimming and filtering of low-quality reads (quality ${ < } 2 0 { \it \Delta }$ ) using Cutadapt $( \mathsf { v } 4 . 1 ) ^ { 9 9 }$ . Clean reads were mapped to the Arabidopsis genome (TAIR10) using HISTA2 (v2.2.1)100. Only uniquely mapped reads were retained for the following analyses. The featureCounts103 was used to obtain raw read counts for all annotated genes. PCA was performed with plotPCA from the DESeq2 R package104. Normalization and modeling of RNA decay data were carried out using the package ‘RNAdecay’ (bioconductor.org/packages/release/bioc/html/RNAdecay.html). Specifically, 30 stably and highly expressed genes in WT and amiR-mta under $+ \mathsf { P }$ and −P conditions (“AT1G44575”, “AT1G30380”, “AT5G42980”, “AT5G65220”, “AT2G20260”, “AT4G01310”, “AT2G37660”, “AT3G01480”, “AT5G08650”, “ATCG00800”, “AT3G02730”, “AT5G59613”, “AT5G38520”, “AT3G11930”, “AT5G41700”, “AT5G03880”, “AT1G76200”, “AT2G44650”, “AT2G42210”, “AT5G16710”, “AT1G02920”, “AT4G20150”, “AT1G21065”, “AT2G21960”, “AT1G20850”, “AT2G24940”, “AT1G66670”, “AT1G15120”, “AT5G16400”, “AT3G12260”) were chosen for data normalization. ComplexHeatmap107 was employed to draw the RNA decay heatmap, which was constructed using the mean relative abundance and half-life of each transcript (Supplementary Data 3).

# Validation of $\mathbf { m } ^ { 6 } \mathbf { A }$ sites in target mRNAs

The PHR1 $\mathsf { m } ^ { 6 } \mathsf { A }$ sites were validated using the SELECT kit (EPIBIOTECH, Cat. No: R202106M-0224T)85. Briefly, 1 μg total RNA of each sample was equally divided into two fractions: one was treated with the $\boldsymbol { \mathsf { m } } ^ { 6 } \boldsymbol { \mathsf { A } }$ demethylase FTO at $3 7 ^ { \circ } \mathrm { C }$ for 30 min, and the other was treated with FTO and EDTA under the same conditions. The probes and primers used are listed in Supplementary Data 5. 

# Gene-specific $\mathbf { m } ^ { 6 } \mathbf { A }$ -IP-qPCR

The gene-specific $\mathsf { m } ^ { 6 } \mathsf { A }$ -IP-qPCR was performed as described in a previous study19. In brief, approximately $2 0 0 { \mathrm { n g } }$ isolated mRNAs was fragmented to 100–150 nt using the Magnesium RNA Fragmentation Module (NEB) and purified by ethanol precipitation. The commercial $\boldsymbol { \mathsf { m } } ^ { 6 } \mathbf { A }$ probes (NEB, Cat# E1610S) and non- $\mathsf { m } ^ { 6 } \mathsf { A }$ probes (NEB, Cat# E1610S) were mixed into the eluted fragmented RNAs as positive and negative controls, respectively. Ten percent of the RNA mixture served as input sample, and the remaining was incubated with $1 \mu |$ anti- $\mathbf { m } ^ { 6 } \mathbf { A }$ antibody (NEB, Cat# E1610S) in $2 0 0 { \mu \mathrm { l } }$ of IP buffer $( 1 0 \mathsf { m } \mathsf { M }$ Tris-HCl pH 7.5, 150 mM NaCl, $0 . 1 \%$ v/v Igepal [CA-6300], and $0 . 2 \mathrm { U } / \mu \mathrm { l }$ SUPERase-In 

RNase Inhibitor [ThermoFisher] in nuclease-free water) at $4 ^ { \circ } \mathrm { C }$ for 2 h on an orbital rotator. The $\mathsf { m } ^ { 6 } \mathsf { A }$ -containing RNA fragments were pulled down and eluted according to the above $\mathsf { m } ^ { 6 } \mathsf { A }$ -seq procedure. Both input and IP samples were analyzed by RT-qPCR using primers listed in Supplementary Data 5. # Measuring phosphate (Pi) content

Measurements of Pi concentrations in plants were carried out as previously described50. In brief, $1 0 { - } 2 0 \mathrm { m g }$ of fresh plant samples were placed in $1 . 5 \mathsf { m l }$ Eppendorf tubes, frozen in liquid nitrogen, and ground to a powder using a Tissuelyser-24 (JINGXIN). After adding $5 0 0 \mu \up$ of $1 \%$ acetic acid, the sample was incubated in the dark at $4 ^ { \circ } \mathrm { C }$ for 30 min with a shaking at 15 min. The mixture was centrifuged at $4 ^ { \circ } \mathrm { C } ,$ $1 2 , 0 0 0 \times g$ for 10 min, and $9 0 \mu \mu \mu$ of the supernatant was transferred to a new $1 . 5 \mathsf { m l }$ Eppendorf tube. The sample was mixed with $2 1 0 \mu \mathrm { l }$ of reaction buffer containing $0 . 4 2 \%$ ammonium molybdate and $10 \%$ ascorbic acid (v/v, 6:1). The sample was incubated at $3 7 ^ { \circ } \mathrm { C }$ for 1 h, and the optical density (OD) at 820 nm was measured in a Spark Multimode Microplate Reader (Tecan). A standard curve was constructed using different concentrations of ${ \mathrm { K H } } _ { 2 } { \mathsf { P O } } _ { 4 }$ (0, 0.5, 1, 2.5, 5, 7.5, 10, 15, 25, 30, 50 nmol), and the Pi concentrations in shoots were calculated. 

# Measuring anthocyanin content

Anthocyanin content was measured as previously described53. After measuring the fresh weight of each sample (typically 0.1–0.5 g), the sample was treated with 1 ml of anthocyanin extraction buffer composed of propano $/ \mathrm { H C l } / \mathrm { H } _ { 2 } \mathrm { O }$ at a ratio of 18/1/81 (v/v/v). Following overnight incubation at room temperature in the dark, the samples were centrifuged at $1 2 , 0 0 0 \times g$ for 20 min. The supernatants were diluted two-fold with extraction buffer, and their absorbance (A) was measured at 535 nm and $6 5 0 \mathsf { n m }$ . The relative anthocyanin content was then determined using the formula: $2 \times ( \mathsf { A } _ { 5 3 5 } - \mathsf { A } _ { 6 5 0 } )$ per gram of fresh weight.
